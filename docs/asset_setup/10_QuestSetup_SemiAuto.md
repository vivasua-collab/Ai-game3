# Настройка квестов (Полуавтомат)

**Инструмент:** Ручная настройка + JSON данные

---

## Внимание!

**Квесты НЕ генерируются автоматически!**

Используй JSON данные как справочник: `Assets/Data/JSON/quests.json`

---

## Шаг 1: Создание QuestData (ВРУКАМИ)

1. `Assets/Data/Quests/` → Create → Cultivation → Quest
2. Назови по шаблону: `Quest_[Name]`

---

## Структура QuestData

| Поле | Тип | Описание |
|------|-----|----------|
| questId | string | Уникальный ID |
| nameRu | string | Название на русском |
| nameEn | string | Название на английском |
| description | text | Описание |
| questType | QuestType | Main, Side, Daily, Cultivation |
| objectives | List | Цели квеста |
| rewards | List | Награды |
| requiredLevel | int | Мин. уровень культивации |
