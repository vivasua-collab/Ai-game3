**Инструмент:** `Tools → Generate Assets → Materials (17)`

---

## Что делает скрипт АВТОМАТИЧЕСКИ:

| Действие | Статус |
|----------|--------|
| Создание папки Materials/ | ✅ Автоматически |
| Чтение JSON данных | ✅ Автоматически |
| Создание MaterialData assets | ✅ Автоматически |
| Заполнение всех полей | ✅ Автоматически |
| Парсинг категорий | ✅ Автоматически |
| Добавление иконок | ❌ Руками |

## Использование материалов

**При крафте:**
```csharp
// Бонусы от материала
float finalDamage = baseDamage * (1 + material.damageBonus);
float finalDefense = baseDefense * (1 + material.defenseBonus);
float qiEfficiency = baseEfficiency + material.qiConductivityBonus;
```
