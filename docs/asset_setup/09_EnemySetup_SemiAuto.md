# Настройка врагов (Полуавтомат)

**Инструмент:** Ручная настройка + JSON данные

---

## Внимание!

**Враги НЕ генерируются автоматически!**

Скрипт `EnemyGenerator` отсутствует. Используй:
1. JSON данные как справочник: `Assets/Data/JSON/enemies.json`
2. Ручное создание префабов в Unity

---

## Шаг 1: Создание префаба врага (ВРУКАМИ)

1. `Assets/Prefabs/Enemies/` → Create → Folder
2. Правый клик → Create → Prefab
3. Назови по шаблону: `Enemy_[Name]`

---

## Шаг 2: Компоненты врага

**Обязательные компоненты:**
```
Enemy:
├── Transform
├── Rigidbody2D (Gravity Scale: 0)
├── CircleCollider2D (Is Trigger: false)
├── SpriteRenderer
├── NPCController (или EnemyController)
├── Combatant
└── HitDetector
```

## Настройка AI

Для врагов используй `NPCAI` с параметрами:
```
NPCAI:
├── behaviorType: Aggressive
├── aggressiveness: 70-100
├── courage: 50-90
└── detectionRange: 5-15
```
