## 📋 Обзор

Тайловая система — основа локальных карт, на которых происходит основной геймплей. Каждый тайл представляет собой ячейку 10×10 м, содержащую данные о проходимости, объектах, субъектах и параметрах окружения.

```
│   │   Z = +1  ████████████  Нижний склон                               │   │
│   │   Z =  0  ────────────  Базовый уровень (равнина, пол)             │   │
│   │   Z = -1  ▓▓▓▓▓▓▓▓▓▓▓▓  Подвал, пещера                             │   │

│   • Определения перепада высот для падений                                 │
│   • Влияния на скорость передвижения                                       │
│   • Бонусов/штрафтов к бою (выше = преимущество)                           │

└─────────────────────────────────────────────────────────────────────────────┘
```

#### Диапазон высот

| Уровень Z | Описание | Примеры |
|-----------|----------|---------|
| **+5 и выше** | Экстремальная высота | Пики гор, небесные острова |
| **+3..+4** | Высокая точка | Вершины холмов, башни |
| **+1..+2** | Возвышенность | Холмы, уступы |
| **0** | Базовый уровень | Равнина, пол, поверхность |
| **-1..-2** | Понижение | Ямы, овраги, подвалы |
| **-3 и ниже** | Глубина | Пропасти, глубокие пещеры |

#### Влияние высоты на геймплей

| Разница Z | Эффект | Механика |
|-----------|--------|----------|
| **+2 и выше** | Падение | Урон = (Z_diff - 1) × 20 |
| **+1** | Преимущество | +10% к урону, +5% к уклонению |
| **0** | Нейтрально | Без бонусов |
| **-1** | Невыгодно | -5% к урону, +5% получить крит |
| **-2 и ниже** | Серьёзный минус | -15% урон, -10% уклонение |

```
│   │   ├── Двери (door)             — Открыть/закрыть                   │   │
│   │   ├── Рудные жилы (ore_vein)   — Добывать                          │   │
│   │   ├── Травы (herb)             — Собирать                          │   │
│   │   └── Места силы (shrine)      — Молиться/медитировать             │   │
│   │                                                                     │   │

│                                                                              │

```

#### Структура данных объекта

| Параметр | Тип | Описание |
|----------|-----|----------|
| `objectId` | string | Уникальный ID объекта |
| `objectType` | enum | Тип объекта (tree, rock, chest...) |
| `isPassable` | bool | Можно ли пройти |
| `moveCost` | float | Множитель стоимости движения (0 = непроходимо) |
| `blocksVision` | bool | Блокирует ли линию видимости |
| `coverBonus` | int | Бонус укрытия (0-100%) |
| `durability` | int | Прочность (если разрушаемое) |
| `interactable` | bool | Можно ли взаимодействовать |
| `interactionType` | enum | Тип взаимодействия |
| `prefabRef` | string | Ссылка на префаб Unity |

#### Множители стоимости движения

| Тип объекта | Move Cost | Примечание |
|-------------|-----------|------------|
| Пустой тайл | 1.0 | Базовая скорость |
| Дорога | 0.7 | Ускорение на 30% |
| Трава | 1.0 | Без изменений |
| Кусты | 1.5 | Замедление 50% |
| Высокая трава | 1.2 | Небольшое замедление |
| Мелкая вода | 2.0 | Сильное замедление |
| Глубокая вода | 0.0 | Непроходимо (требует навык) |
| Горы | 3.0 | Очень медленно |
| Лёд | 1.5 + скольжение | Особая механика |

#### Расчёт плотности Ци на тайле

```
finalQiDensity = baseQiDensity
               × terrainModifier
               × climateModifier
               × localSourceModifier
               × formationModifier
               × eventModifier
```

## ⚡ Оптимизация

### Стратегии оптимизации

| Техника | Описание | Применение |
|---------|----------|------------|
| **Chunk Loading** | Загрузка только видимых чанков | Большие локации |
| **Delta Compression** | Сохранение только изменений | Уменьшение размера |
| **LOD (Level of Detail)** | Упрощение далёких тайлов | Визуализация |
| **Spatial Hashing** | Быстрый поиск по координатам | Запросы объектов |
| **Dirty Flag** | Отложенное обновление | Производительность |
