## 📋 Полный список Skills

### 🤖 AI & Media Processing

---

#### 1. ASR (Automatic Speech Recognition)

| Возможность | Описание |
|-------------|----------|
| 📝 Транскрибация | Преобразование аудиофайлов в текст |
| 🎤 Речь → текст | Распознавание речи в реальном времени |
| 🔊 Аудио обработка | Работа с различными аудиоформатами |

**Использование:** `Skill(command="ASR")`

---

#### 2. TTS (Text-to-Speech)

| Возможность | Описание |
|-------------|----------|
| 🔊 Синтез речи | Преобразование текста в естественную речь |
| 🎵 Аудиоконтент | Создание аудиофайлов из текста |
| 🗣️ Голос | Генерация голосовых файлов |

**Использование:** `Skill(command="TTS")`

---

#### 3. LLM (Large Language Model)

| Возможность | Описание |
|-------------|----------|
| 💬 Чат-боты | Создание AI-ассистентов и ботов |
| 🤥 Многотуровые разговоры | Поддержание контекста диалога |
| 📚 Контекст | Управление историей разговора |

**Использование:** `Skill(command="LLM")`

---

#### 4. VLM (Vision Language Model)

| Возможность | Описание |
|-------------|----------|
| 🖼️ Анализ изображений | Понимание визуального контента |
| 📄 Документы | Работа с отсканированными документами |
| 🎨 Мультимодальность | Одновременная работа с текстом и изображениями |

**Использование:** `Skill(command="VLM")`

---

#### 5. Image-Generation

| Возможность | Описание |
|-------------|----------|
| 🎨 Генерация | Создание изображений из текстовых описаний |
| 🖌️ Арт | Создание artwork и дизайн-ассетов |
| 📸 Визуал | Генерация визуального контента для игр/UI |

**Использование:** `Skill(command="Image-Generation")`

**Для проекта Unity:**
- Генерация иконок для предметов
- Создание концепт-арта персонажей
- UI элементы и текстуры

---

#### 6. Video-Generation

| Возможность | Описание |
|-------------|----------|
| 🎬 Генерация видео | Создание видео из текстовых промптов |
| 📹 Видеоконтент | AI-powered создание видеороликов |
| 🎥 Трейлеры | Создание промо-материалов |

**Использование:** `Skill(command="Video-Generation")`

---

#### 7. Video-Understanding

|-------------|----------|

**Использование:** `Skill(command="Video-Understanding")`

## 📝 Примеры использования

### Поиск информации о Unity

```
1. Web-Search: "Unity 6.3 DOTS best practices"
2. Web-Reader: https://docs.unity3d.com/...
3. Синтез информации в документ
```

### Генерация ассетов

```
1. Image-Generation: "Icon for cultivation technique, fire element, Chinese style"
2. Сохранение результата
3. Интеграция в Unity проект
```

### Создание баланса

```
1. XLSX: Создать таблицу техник
2. Заполнить параметры (урон, стоимость, эффекты)
3. Экспорт в JSON для Unity
```
