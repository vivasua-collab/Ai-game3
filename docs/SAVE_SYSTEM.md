## 🔄 Автосохранение

### Триггеры

| Событие | Действие |
|---------|----------|
| Смена локации | Сохранить текущее состояние |
| Получение техники | Сохранить персонажа |
| Получение важного предмета | Сохранить инвентарь |
| Прорыв уровня | Сохранить персонажа |
| Завершение боя | Сохранить состояние боя |
| Каждые 60 тиков | Периодическое сохранение |

### Ручное сохранение

Игрок может сохранить игру в любой момент через меню.

## 📦 Структура файла сохранения

### Данные персонажа

- Характеристики (STR, AGI, INT, VIT)
- Уровень культивации
- Текущее и максимальное Ци (**тип `long`** — с Fix-01)
- Проводимость
- Состояние тела (части тела, HP)

### Данные техник

- Изученные техники
- Мастерство каждой техники
- Назначение в слоты

### Данные инвентаря

- Все предметы с характеристиками
- Экипировка в слотах
- Духовное хранилище

### Данные формаций и баффов (добавлено в Fix-08)

- FormationSaveData — активные формации, позиция, стадия, пул Ци
- BuffSaveData — активные баффы, оставшееся время, стеки

### Данные зарядника (добавлено в Fix-08)

- ChargerSaveData — слоты, вставленные камни, тепло, буфер

### Данные тайлов (добавлено в Fix-08)

- TileSaveData — изменённые тайлы, объекты, температура

### Данные NPC (дополнено в Fix-07)

- NPCSaveData — Attitude (числовое -100..+100 вместо Disposition) — согласовано с NPC_AI_SYSTEM.md Fix-07
- PersonalityTrait [Flags] — черты характера (Aggressive, Cautious, Treacherous, Ambitious, Loyal, Pacifist, Curious, Vengeful)
- SkillLevelData — навыки NPC по уровням

### Данные мира

- Текущее время
- Текущая локация
- Состояние NPC
- Выполненные квесты
