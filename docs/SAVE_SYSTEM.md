## 📋 Обзор

Система сохранения управляет сохранением и загрузкой игрового состояния.

---

## 🔄 Автосохранение

### Триггеры

| Событие | Действие |
|---------|----------|
| Смена локации | Сохранить текущее состояние |
| Получение техники | Сохранить персонажа |
| Получение важного предмета | Сохранить инвентарь |
| Прорыв уровня | Сохранить персонажа |
| Завершение боя | Сохранить состояние боя |
| Каждые 60 тиков | Периодическое сохранение |

### Ручное сохранение

Игрок может сохранить игру в любой момент через меню.

---

## 📦 Структура файла сохранения

### Данные персонажа

- Характеристики (STR, AGI, INT, VIT)
- Уровень культивации
- Текущее и максимальное Ци
- Проводимость
- Состояние тела (части тела, HP)

### Данные техник

- Изученные техники
- Мастерство каждой техники
- Назначение в слоты

### Данные инвентаря

- Все предметы с характеристиками
- Экипировка в слотах
- Духовное хранилище

### Данные мира

- Текущее время
- Текущая локация
- Состояние NPC
- Выполненные квесты
