## 1. Обзор системы переходов

### 1.1 Концепция

Система переходов управляет перемещением игрока между различными уровнями детализации мира: от глобальной карты мира к конкретным локациям и интерьерам зданий.

### 1.2 Двухуровневая модель

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                      УРОВЕНЬ 1: МИРОВАЯ КАРТА                                │
│                                                                              │
│    ┌────┬────┬────┬────┬────┐                                              │
│    │ 🏔️ │ 🌲 │ 🏘️ │ 🌲 │ 🌊 │   Игрок видит мир "с высоты птичьего полёта" │
│    ├────┼────┼────┼────┼────┤   Перемещение между секторами                 │
│    │ 🌲 │ 🏛️ │ 👤 │ 🌾 │ 🌲 │   Время течёт ускоренно                       │
│    ├────┼────┼────┼────┼────┤                                                │
```

### 1.4 Виды переходов

| Тип перехода | Направление | Триггер | Время загрузки |
|--------------|-------------|---------|----------------|
| World → Location | Вглубь | Клик/вход в сектор | Среднее |
| Location → World | Наружу | Выход/телепорт | Быстрое |
| Location → Building | Вглубь | Вход в дверь | Быстрое |
| Building → Location | Наружу | Выход из двери | Быстрое |
| Телепортация | Любой | Техника/предмет | Мгновенное |
| Измерение | Спец. | Формация/ритуал | Долгое |

```
│                                                                              │
```


### 2.5 Время путешествия

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                       ВРЕМЯ ПУТЕШЕСТВИЯ ПО КАРТЕ                             │

│   │   gameTime = distance × terrainMultiplier / speedMultiplier          │    │
│   │                                                                      │    │
│   │   Где:                                                               │    │
│   │   - distance — расстояние в километрах                              │    │
```
