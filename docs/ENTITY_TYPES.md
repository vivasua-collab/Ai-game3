## 2️⃣ Уровень 1: SoulType (Природа сущности)

### Типы SoulType

| SoulType | Body | Qi | Mind | Описание | Примеры |
|----------|------|-----|------|----------|---------|
| `character` | ✅ organic | ✅ core | ✅ full | Разумные существа | Игрок, NPC, Эльф, Демон |
| `creature` | ✅ organic/scaled | ✅ core | ⚠️ instinct | Животные, звери | Волк, Тигр, Дракон |
| `spirit` | ❌ / ethereal | ✅ reservoir | ✅ full | Бесплотные сущности | Призрак, Элементаль |
| `artifact` | ✅ mineral | ✅ reservoir | ⚠️ simple | Разумные предметы | Разумный меч |
| `construct` | ✅ construct/mineral | ✅ reservoir | ⚠️ simple | Искусственные создания | Голем |

### Характеристики по типам

**Character:**
- Полное сознание и речь
- Может культивировать (ядро)
- Может использовать техники
- Социальные взаимодействия

**Creature:**
- Инстинктивное поведение
- Может культивировать (ядро)
- Врождённые техники
- Не может говорить (обычно)

**Spirit:**
- Полное сознание
- Резервуар Ци (не ядро)
- Эфирное тело
- Иммунитет к физике

**Artifact:**
- Простое сознание
- Резервуар Ци
- Материальное тело
- Специфические функции

**Construct:**
- Простое сознание
- Резервуар Ци
- Искусственное тело
- Следует командам

## 3️⃣ Уровень 2: Morphology (Морфология)

### Типы морфологии

| Morphology | Описание | Части тела | Примеры |
|------------|----------|------------|---------|
| `humanoid` | Двурукое двуногое | 11 + сердце | Человек, Эльф, Демон |
| `quadruped` | Четвероногое | 8 + сердце | Волк, Тигр, Медведь |
| `bird` | Крылатое | 6-7 + сердце | Орёл, Феникс |
| `serpentine` | Змееподобное | 6 + сегменты | Змея, Ламия |
| `arthropod` | Членистоногое | Экзоскелет | Паук, Скорпион |
| `amorphous` | Бесформенное | 2 (core + essence) | Призрак, Элементаль |
| `hybrid_centaur` | Кентавр | 12 + сердце | Кентавр |
| `hybrid_mermaid` | Русалка | 8 + сердце | Русалка |
| `hybrid_harpy` | Гарпия | 9 + сердце | Гарпия |
| `hybrid_lamia` | Ламия | 8 + сегменты | Ламия |

### Соответствие SoulType → Morphology

| SoulType | Допустимые Morphology |
|----------|----------------------|
| `character` | humanoid, hybrid_* |
| `creature` | quadruped, bird, serpentine, arthropod |
| `spirit` | amorphous (всегда) |
| `artifact` | humanoid (редко), amorphous |
| `construct` | humanoid, quadruped |

## 5️⃣ Материалы тела

### Типы материалов

| Материал | Твёрдость | Снижение урона | Особенности |
|----------|-----------|----------------|-------------|
| `organic` | 3 | 0% | Базовый материал |
| `scaled` | 6 | 30% | Чешуя, защита от физики |
| `chitin` | 5 | 20% | Экзоскелет, уязвим к дробящему |
| `ethereal` | 1 | 70% физики | Проходит сквозь материю |
| `mineral` | 8 | 50% | Камень, очень твёрдый |
| `construct` | 5-8 | 30-50% | Зависит от материалов |
| `chaos` | 5 | Переменно | Непредсказуемые свойства |

### Влияние на HP

Материал увеличивает итоговое HP частей тела:
- Organic: базовый множитель (×1.0)
- Scaled: +30% HP
- Chitin: +20% HP
- Mineral: +50% HP

## 6️⃣ SoulEntity (Живые объекты)

### Архитектура

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          SOUL ENTITY (ЖИВАЯ СУЩНОСТЬ)                   │
├─────────────────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │                       SOUL CORE (ОБЯЗАТЕЛЬНО)                    │   │
│  │  id | soulType | controller (player/ai) | status                │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                              │                                           │
│              ┌───────────────┼───────────────┐                          │
│              ▼               ▼               ▼                          │
│  ┌───────────────┐  ┌───────────────┐  ┌───────────────┐               │
│  │     BODY      │  │      QI       │  │    MIND       │               │
│  │  (опционально)│  │  (обязательно)│  │  (опционально)│               │
│  └───────────────┘  └───────────────┘  └───────────────┘               │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  ATTACHMENTS: techniques[] | inventory[] | effects[]             │   │
│  └─────────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────────┘
```

### Компоненты

**Soul Core (обязательно):**
- Уникальный идентификатор
- Тип сущности (SoulType)
- Контроллер (player или ai)
- Статус (alive, dead, unconscious)

**Body (опционально):**
- Физическое тело
- Части тела с HP
- Материал тела
- Морфология

**Qi (обязательно):**
- Ядро (для character, creature) или резервуар (для spirit, construct)
- Текущее и максимальное Ци
- Проводимость

**Mind (опционально):**
- Тип сознания (full, instinct, simple)
- Личность (для character)
- Память и знания

│  └─────────────────────────────────────────────────────────────────┘   │
│                              │                                           │
│              ┌───────────────┼───────────────┐                          │
│              ▼               ▼               ▼                          │
│  ┌───────────────┐  ┌───────────────┐  ┌───────────────┐               │
│  │    PHYSICS    │  │   DURABILITY  │  │   QI STORE    │               │
│  │  (обязательно)│  │   (опц.)      │  │   (опц.)      │               │
│  └───────────────┘  └───────────────┘  └───────────────┘               │
│  ┌─────────────────────────────────────────────────────────────────┐   │

### Категории

| Категория | Взаимодействие | Примеры |
|-----------|----------------|---------|
| `resource` | mine, harvest, chop | Руда, дерево, трава |
| `structure` | — | Стены, здания |
| `container` | open | Сундуки |
| `decoration` | — | Камни, статуи |
| `item` | use | Предметы |

## 8️⃣ Сравнение SoulEntity vs PhysicalObject

| Критерий | SoulEntity | PhysicalObject |
|----------|------------|----------------|
| **Душа** | ✅ Есть | ❌ Нет |
| **Контроллер** | player / ai | none |
| **Действия** | Осознанные | Реактивные |
| **Движение** | Самостоятельное | Неподвижны |
| **Qi** | core или reservoir | qiStore (опц.) |
| **Техники** | ✅ Используют | ❌ Не могут |
| **Инвентарь** | ✅ Носит | ✅ Может содержать |
| **Развитие** | ✅ Культивация | ❌ Нет |
