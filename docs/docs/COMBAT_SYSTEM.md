## 💥 Расчёт урона

### Базовый урон техники

**Формула:**
```
finalDamage = capacity × gradeMult
```

Где:
- `capacity` = затраченное Ци (ограничено ёмкостью техники)
- `gradeMult` = множитель Grade (×1.0 ~ ×1.6)

### Модификаторы урона

| Модификатор | Формула |
|-------------|---------|
| Мастерство | +0.5% за каждый % мастерства |
| Проводимость | +0.3% за каждую единицу проводимости |
| Уровень культивации | × множитель уровня |
| Стихийный бонус | Зависит от стихии |

## 📊 Формула Qi Density (Качество Ци)

> **Источник истины:** [ALGORITHMS.md](./ALGORITHMS.md) §3.3 "Плотность Ци"

**Принцип:** Качество Ци = qiDensity = 2^(уровень-1)

Геометрический рост ×2 за каждый уровень культивации. Подробная таблица — в ALGORITHMS.md.

### Полная формула урона техники

```
rawDamage = qiInvested × qiDensity × statMultiplier × masteryBonus × elementalBonus
```

Где:
- `qiInvested` — вложенное Ци (базовое или масштабированное)
- `qiDensity` = 2^(cultivationLevel-1)
- `statMultiplier` — масштабирование от характеристик (STR/AGI/INT)
- `masteryBonus` — бонус мастерства (до +30% при 100%)
- `elementalBonus` — стихийный модификатор

### Масштабируемая стоимость техник

Игрок может вложить больше Ци для усиления эффекта:

| Параметр | Описание |
|----------|----------|
| baseQiCost | Минимальная стоимость техники |
| maxQiMultiplier | Максимальный множитель (напр. 10x) |

**Пример:**
- Техника "Огненный шар": baseQiCost=50, maxQiMultiplier=10
- L1 практик, вложил 50 Ци: Урон = 50 × 1 × 1 = 50
- L9 практик, вложил 500 Ци: Урон = 50 × 10 × 256 = 128,000

## 📊 Подавление уровнем (Level Suppression)

> **Источник истины:** [ALGORITHMS.md](./ALGORITHMS.md) §1 "Система подавления уровнем"

**Проблема:** Qi Buffer с поглощением 90% позволяет практике L1 наносить урон практике L8.

**Решение:** Множитель подавления на основе разницы уровней.

**Кратко:**
| Разница | Normal | Technique | Ultimate |
|---------|--------|-----------|----------|
| 0 | ×1.0 | ×1.0 | ×1.0 |
| 1 | ×0.5 | ×0.75 | ×1.0 |
| 2 | ×0.1 | ×0.25 | ×0.5 |
| 3 | ×0.0 | ×0.05 | ×0.25 |
| 5+ | ×0.0 | ×0.0 | ×0.0 |

Полная таблица и формулы — в ALGORITHMS.md.

## 🩸 Применение урона к телу

### Поток повреждения

1. **Функциональная HP** снижается первой
2. При 0 функциональной → урон идёт в структурную
3. При 0 структурной → часть отрублена

### Кровотечение

При значительном уроне начинается кровотечение:

| Степень | HP/тик |
|---------|--------|
| Лёгкое | 1 |
| Среднее | 3 |
| Тяжёлое | 5 |
| Критическое | 10 |
