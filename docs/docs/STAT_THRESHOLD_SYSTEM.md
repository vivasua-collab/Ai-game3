# 📊 Система порогов развития характеристик

**Версия:** 1.0  
**Дата:** 2026-03-31  
**Статус:** 📋 Теоретический документ

---

## ⚠️ Важно

> **Принцип:** "Нет стадии куколки" — развитие всегда постепенное.  
> Чем выше характеристика, тем больше усилий требуется для повышения.

---

## 📋 Обзор

Система порогов определяет, сколько усилий (виртуальной дельты) нужно для повышения каждой характеристики на 1 единицу.

---

## 🎯 Основные принципы

### 1. Линейное развитие

**Из лора:** "нет стадии куколки, чтобы резко делать скачки"

**Решение:**
- Постепенный прирост за действия
- Накопление в виртуальной дельте
- Закрепление только при сне
- Пороги растут с характеристикой

### 2. Четыре основные характеристики

| Характеристика | Описание | Влияние |
|----------------|----------|---------|
| **Strength (STR)** | Физическая сила | Урон, переносимый вес |
| **Agility (AGI)** | Ловкость | Уклонение, скорость атаки |
| **Intelligence (INT)** | Интеллект | Эффективность техник, обучение |
| **Vitality (VIT)** | Живучесть | HP частей тела, регенерация |

---

## 📐 Формула порога

### Базовая формула

```
threshold = floor(currentStat / 10)
```

**Примеры:**
| Текущий стат | Порог для +1 | Пример |
|--------------|--------------|--------|
| 10 | 1.0 | 10 → 11: нужно 1.0 дельты |
| 15 | 1.5 | 15 → 16: нужно 1.5 дельты |
| 20 | 2.0 | 20 → 21: нужно 2.0 дельты |
| 50 | 5.0 | 50 → 51: нужно 5.0 дельты |
| 100 | 10.0 | 100 → 101: нужно 10.0 дельты |

### Модификаторы

| Фактор | Множитель |
|--------|-----------|
| Базовый | ×1.0 |
| Техника ускорения | ×1.5 |
| Артефакт обучения | ×1.3 |
| Возраст < 20 | ×1.2 |
| Возраст > 50 | ×0.8 |

---

## 💫 Виртуальная дельта

### Концепция

Виртуальная дельта — накопленный прогресс, который ещё не закреплён в реальной характеристике.

```
┌─────────────────────────────────────────────────────────────────┐
│                    СИСТЕМА ВИРТУАЛЬНОЙ ДЕЛЬТЫ                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   Физическая нагрузка (тренировка/бой)                          │
│       │                                                          │
│       ▼                                                          │
│   ┌───────────────────────────────────────────────┐             │
│   │ ВИРТУАЛЬНАЯ ДЕЛЬТА (временная)                │             │
│   │                                               │             │
│   │ virtualStrengthDelta: +0.047                  │             │
│   │ virtualAgilityDelta: +0.023                   │             │
│   │                                               │             │
│   │ НЕ влияет на боевые характеристики!           │             │
│   │ НЕ теряется без сна (кап на дельту)           │             │
│   └───────────────────────────────────────────────┘             │
│       │                                                          │
│       │ Минимум 4 часа сна                                       │
│       ▼                                                          │
│   ┌───────────────────────────────────────────────┐             │
│   │ ЗАКРЕПЛЕНИЕ (конвертация)                     │             │
│   │                                               │             │
│   │ Сон 4+ часов:                                │             │
│   │ - Максимум за сон: +0.20 (8 часов)           │             │
│   │ - Остаток сохраняется в виртуальной дельте   │             │
│   └───────────────────────────────────────────────┘             │
│       │                                                          │
│       ▼                                                          │
│   ┌───────────────────────────────────────────────┐             │
│   │ РЕАЛЬНАЯ ХАРАКТЕРИСТИКА                       │             │
│   │                                               │             │
│   │ strength: 12.047 → 12.067 (закреплено +0.02)  │             │
│   └───────────────────────────────────────────────┘             │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Капы виртуальной дельты

| Характеристика | Максимум дельты |
|----------------|-----------------|
| Strength | 10.0 |
| Agility | 10.0 |
| Intelligence | 15.0 |
| Vitality | 10.0 |

## 🎮 Источники прироста

### Боевые действия

| Действие | Прирост | Характеристика |
|----------|---------|----------------|
| Удар в бою | +0.001 | Strength |
| Уклонение | +0.001 | Agility |
| Блок | +0.0005 | Strength |
| Получение урона | +0.001 | Vitality |
| Использование техники | +0.0005 | Intelligence |

### Тренировки

| Действие | Прирост | Характеристика |
|----------|---------|----------------|
| Физическая тренировка | +0.002/мин | По выбору |
| Спарринг | +0.003/мин | STR + AGI |
| Медитация | +0.01/мин | Intelligence |
| Закалка тела | +0.005/мин | Vitality |

### Лечение и восстановление

| Действие | Прирост | Характеристика |
|----------|---------|----------------|
| Получение урона + лечение | +0.005 | Vitality |
| Регенерация конечности | +0.01 | Vitality |
| Лекарственные ванны | +0.02 | Vitality |

## 😴 Закрепление при сне

### Правила

1. **Минимум 4 часа** сна для начала закрепления
2. **Максимум закрепления за сон:** +0.20 (при 8 часах)
3. Остаток дельты сохраняется

### Формула закрепления

```
consolidated = min(virtualDelta, sleepHours * 0.025)
```

### Таблица закрепления

| Часы сна | Закрепление | Пример |
|----------|-------------|--------|
| 4 часа | +0.067 | ~1/15 дельты |
| 5 часов | +0.100 | ~1/10 дельты |
| 6 часов | +0.133 | ~1/7 дельты |
| 7 часов | +0.167 | ~1/6 дельты |
| 8 часов | +0.200 | ~1/5 дельты |

## 📊 Достижимость за периоды

### Расчёт времени

**Предположения:**
- Ежедневная тренировка: 2 часа
- Ежедневный бой: 30 минут
- Сон: 8 часов

### Таблица достижимости

| Период | Достижимый стат | Время на +1 (при стате 20) |
|--------|-----------------|----------------------------|
| 100 дней | ~25 | 15-20 дней |
| 365 дней (1 год) | ~35 | 20-25 дней |
| 1000 дней (~3 года) | ~55 | 21-28 дней |
| 3000 дней (~8 лет) | ~80 | 42-56 дней |
| 10000 дней (~27 лет) | ~125 | 70+ дней |

### Структура данных StatDevelopment

Ключевые поля:
- **Реальные характеристики:** `strength`, `agility`, `intelligence`, `vitality`
- **Виртуальные дельты:** `virtualStrengthDelta`, `virtualAgilityDelta`, `virtualIntelligenceDelta`, `virtualVitalityDelta`

Ключевые методы:
- `AddDelta(StatType type, float amount)` — добавление прогресса
- `GetThreshold(StatType type)` — расчёт порога
- `ConsolidateSleep(float hours)` — закрепление при сне
- `CanAdvance(StatType type)` — проверка возможности повышения

```
threshold = floor(currentStat / 10)
```

Примеры:

|------|-------|----------|
| 10 | 1.0 | 10 → 11: нужно 1.0 дельты |
| 20 | 2.0 | 20 → 21: нужно 2.0 дельты |
| 50 | 5.0 | 50 → 51: нужно 5.0 дельты |
| 100 | 10.0 | 100 → 101: нужно 10.0 дельты |

## 🎯 Баланс

### Мягкие капы

Система порогов создаёт "мягкий кап" — теоретически можно развить до бесконечности, но время растёт экспоненциально.

### Примеры времени

| Старт | Цель | Время |
|-------|------|-------|
| 10 | 20 | ~100 дней |
| 20 | 40 | ~500 дней |
| 40 | 60 | ~800 дней |
| 60 | 80 | ~1500 дней |
| 80 | 100 | ~3000 дней |
