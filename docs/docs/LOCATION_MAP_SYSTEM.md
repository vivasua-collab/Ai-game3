## 📋 Обзор

Система локаций описывает процедурную генерацию локальных карт, на которых происходит основной геймплей — исследование, бой, взаимодействие с NPC.

| **Серебряная жила** | Серебро | 2 | Кирка L2+ | 60 сек |

| Стадия | Остаток | Вид | Добыча |

| **Истощённое** | 25-10% | Следы руды | -60% скорость |
| **Выработанное** | <10% | Пустая порода | 0 (визуально пусто) |

## 💡 Предложения и идеи

### Уникальные механики

1. **Живой мир** — NPC имеют расписание, перемещаются, взаимодействуют
2. **Следы** — остающиеся следы игрока (для отслеживания врагами)
3. **Эхо событий** — последствия битв, пожаров видны длительное время
4. **Миграция** — животные мигрируют между локациями по сезонам
5. **Зарастание** — заброшенные локации постепенно покрываются растительностью

### Интересные особенности

1. **Аномальные зоны** — участки с изменённой физикой (антигравитация, замедление времени)
2. **Духовные разломы** — временные проходы в другие измерения
3. **Живая природа** — деревья растут, реки меняют русло
4. **Археологические слои** — под современными постройками — древние руины
5. **Динамические квесты** — события происходят без игрока (город атакован, торговец ограблен)

### Расширения

1. **Подземный мир** — целая система пещер под поверхностью
2. **Духовный мир** — параллельное измерение для духовных практиков
3. **Небесные острова** — летающие локации для L7+
4. **Подводные области** — для техник дыхания под водой
5. **Портальные камеры** — комнаты, связанные телепортами
