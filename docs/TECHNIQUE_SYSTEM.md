## 🎯 Ключевые принципы

### Принцип 1: Урон = Ёмкость × Grade

> **Это главное правило расчёта урона техник.**

```
finalDamage = capacity × gradeMultiplier × ultimateMultiplier
```

Где:
- `gradeMultiplier` = множитель Grade (common:1.0, refined:1.3, perfect:1.6, transcendent:2.0)
- `ultimateMultiplier` = isUltimate ? 2.0 : 1.0

### Принцип 2: Архитектура "Матрёшка"

Три слоя генерации техники:
1. **База** — qiCost, capacity, baseDamage
2. **Grade** — множители качества
3. **Бонусы** — эффекты от Grade и стихии

### Принцип 3: Система тиков

Все эффекты измеряются в тиках (1 тик = 1 минута игрового времени)

## 📊 Система Grade (Качество)

### Уровни Grade

| Grade | Урон | Бонусов | Шанс эффекта |
|-------|------|---------|--------------|
| Common | ×1.0 | 0 | 0% |
| Refined | ×1.3 | 1 | 20% |
| Perfect | ×1.6 | 2 | 50% |
| Transcendent | ×2.0 | 3 | 80% |

**Важно:** Стоимость Ци всегда ×1.0 — не зависит от Grade.

### Распределение Grade

| Grade | Шанс | Примечание |
|-------|------|------------|
| Common | 60% | Базовый |
| Refined | 28% | Улучшенный |
| Perfect | 10% | Редкий |
| Transcendent | 2% | Легендарный (даже на L1!) |

## 🏗️ Структурная ёмкость (Capacity)

### Принцип

Ёмкость техники — максимальное базовое Ци, которое техника может обработать.

### Формула

```
capacity = baseCapacity(type) × 2^(level-1) × (1 + mastery/100 × 0.5)
```

### Базовая ёмкость по типам

| Тип техники | baseCapacity |
|-------------|--------------|
| Formation | 80 |
| Defense | 72 |
| Support | 56 |
| Healing | 56 |
| Combat (melee_strike) | 64 |
| Combat (melee_weapon) | 48 |
| Combat (ranged_*) | 32 |
| Movement | 40 |
| Curse | 40 |
| Poison | 40 |
| Sensory | 32 |
| Cultivation | null (пассивная) |

### Примеры расчёта

**melee_strike L5, mastery 0%:**
```
capacity = 64 × 2^4 × 1.0 = 1024 базового Ци
finalDamage = 1024 × gradeMultiplier × ultimateMultiplier
```

**ranged_projectile L9, mastery 100%:**
```
capacity = 32 × 256 × 1.5 = 12288 базового Ци
```

## 💥 Дестабилизация

### Принцип неделимости Ци

**Кратность выпуска Ци практиком = 1**

Практик выпускает Ци минимальными единицами своего уровня.

| Уровень | 1 единица Ци практика = базового Ци |
|---------|-----------------------------------|
| L1 | 1 |
| L5 | 16 |
| L9 | 256 |

### Переполнение

При `вливаемое Ци > ёмкость техники` → **ПЕРЕПОЛНЕНИЕ**

**Пример:**
- Практик L9 использует технику L2 (capacity = 128)
- 1 единица Ци L9 = 256 базового Ци
- Переполнение: 256 - 128 = 128 единиц

**Результат:**
1. Излишки Ци (128) рассеиваются
2. Урон практику = 128 × 0.5 = 64
3. Урон по цели (только melee!) = 256 × 0.5 = 128

### Правило

> Переполнение ВОЗМОЖНО, но с последствиями.

Для ranged атак: Ци разлетается во все стороны, урона по цели НЕТ.

## 🔒 Ограничения по уровню

### Почему нельзя использовать техники ВЫШЕ своим уровнем?

**Лор: Резонанс Ци**

Техника требует определённой плотности Ци для поддержания структуры канала.

> **Источник истины:** [ALGORITHMS.md](./ALGORITHMS.md) §3.3 "Плотность Ци"

**Минимальный уровень техники:** Практик может использовать техники с L(min) = max(1, L(практик) - 4)

| Уровень практика | qiDensity | Мин. L техники | Макс. L техники |
|------------------|-----------|----------------|-----------------|
| L1 | 1 | L1 | L1 |
| L3 | 4 | L1 | L3 |
| L5 | 16 | L2 | L5 |
| L7 | 64 | L3 | L7 |
| L9 | 256 | L4 | L9 |

| Стихия | ID | Характер |
|--------|-----|----------|

## 🎮 Типы техник

### Combat (Боевые)

| Подтип | Бонус от силы | Бонус от ловкости |
|--------|---------------|-------------------|
| melee_strike | 5% | 2.5% |
| melee_weapon | 2.5% | 5% |
| ranged_projectile | — | 2.5% + INT 5% |
| ranged_beam | — | 2.5% + INT 5% |
| ranged_aoe | — | 2.5% + INT 5% |

### Defense (Защитные)

| Подтип | Механика |
|--------|----------|
| shield | Поглощение урона за счёт Ци |
| block | Снижение урона на % |
| dodge | Шанс уклонения |
| reflect | Отражение урона |

### Support (Поддержка)

> **Важно:** Баффы НЕ увеличивают телесные характеристики!

| Тип | Что баффает |
|-----|-------------|
| Damage | +% урона техник |
| Defense | +% сопротивления |
| Speed | +% передвижения |
| Crit | +% крит шанс |

### Healing (Исцеление)

- **element = 'neutral' ВСЕГДА**
- **БЕЗ стихийных бонусов**
- Эффективность зависит только от Grade и уровня

### Cultivation (Культивация)

- **element = 'neutral' ВСЕГДА**
- **capacity = null** (пассивная)
- **qiCost = 0**
- Работает во время медитации/отдыха

### Curse (Проклятия)

- Длительность: секунды-минуты (боевые) или часы-месяцы (ритуальные)
- Типы: weakness, slowness, silence, exhaustion, soul_burn

### Poison (Яды)

- **element = 'poison' ТОЛЬКО**
- Несколько дебаффов по Grade
- Способы доставки: ingestion, contact, injection, inhalation, technique

### Movement (Перемещение)

| Тип | Описание |
|-----|----------|
| Dash | Быстрый рывок |
| Dodge | Уклонение + неуязвимость |
| Teleport | Мгновенное перемещение |
| Flight | Поддержание полёта |

### Sensory (Восприятие)

| Тип | Описание |
|-----|----------|
| Detection | Обнаружение живых существ |
| Analysis | Информация о цели |
| Vision | Видение сквозь препятствия |
| Link | Ментальная связь |

## 🎯 Бонусы техник

### Три уровня бонусов

**Бонус 1: Сила эффекта от Grade**
- Common: 0%
- Refined: 50%
- Perfect: 100%
- Transcendent: 150%

**Бонус 2: Эффект от стихии**
- Определяется стихией и типом техники

**Бонус 3: Transcendent-эффект**
- Только для Transcendent Grade
- Уникальное свойство стихии
