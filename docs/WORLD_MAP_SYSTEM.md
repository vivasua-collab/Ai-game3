## 1. Обзор Мировой Карты

### 1.1 Концепция двухуровневой системы

Мировая карта реализована как **двухуровневая система навигации**, обеспечивающая баланс между стратегическим обзором и тактическим взаимодействием.

### 2.1 Секторы (Tiles) — Базовые единицы

**Определение:** Сектор — это минимальная единица навигации на карте мира.

```

│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │                                                                      │   │
│   │   ┌─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┐                │   │
│   │   │ 🌲 │ 🌲 │ 🪨 │     │     │ 🌲 │ 🌲 │ 🌲 │                │   │
│   │   ├─────┼─────┼─────┼─────┼─────┼─────┼─────┼─────┤                │   │
```

**Характеристики сектора:**

| Параметр | Тип | Описание |
|----------|-----|----------|
| `sectorId` | string | Уникальный идентификатор (X_Y) |
| `terrainType` | enum | Основной тип местности |
| `elevation` | float | Высота над уровнем моря (0-10000) |
| `qiDensity` | int | Плотность Ци в секторе |
| `dangerLevel` | int | Уровень опасности (1-9) |
| `climate` | enum | Климатическая зона |
| `discovered` | bool | Открыт ли сектор |
| `lastVisited` | long | Timestamp последнего посещения |

**Типы регионов:**

| Тип | Описание | Особенности |
|-----|----------|-------------|
| `Wilderness` | Дикая местность | Случайные враги, ресурсы |
| `Civilized` | Цивилизованная | Города, дороги, торговля |
| `Sacred` | Священная | Высокое Ци, храмы, духи |
| `Cursed` | Проклятая | Низкое Ци, демоны, ловушки |
| `Contested` | Оспариваемая | Войны фракций, столкновения |
| `Restricted` | Ограниченная | Требует разрешения для входа |

### 5.2 Время Путешествия

**Формула расчёта:**

```
travelTime = distance / (speed × terrainModifier × weatherModifier)

Где:
• distance — расстояние в километрах
• speed — базовая скорость (5 км/ч пешком)
• terrainModifier — модификатор местности (0.3-1.0)
• weatherModifier — модификатор погоды (0.5-1.2)

Примеры:
• 10 км по равнине пешком = 10 / (5 × 1.0 × 1.0) = 2 часа
• 10 км по горам пешком = 10 / (5 × 0.3 × 1.0) = 6.7 часа
• 10 км по равнине верхом = 10 / (20 × 1.0 × 1.0) = 30 минут
```
