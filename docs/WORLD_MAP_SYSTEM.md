```
┌─────────────────────────────────────────────────────────────────────────────┐

│   ❌ 1×1 м — СЛИШКОМ МЕЛКО:                                                 │

│   ├── Перерасход памяти и производительности                                │

│   ├── Неточная боевка и позиционирование                                    │
│   ├── Трудно разместить объекты детально                                    │
│   └── Игрок = 1 тайл, дом = 1 тайл (нереалистично)                          │
│                                                                              │
│   ✅ 2×2 м — ОПТИМАЛЬНО:                                                    │
│   ├── Детализация: дерево = 1 тайл, дом = 9-25 тайлов                       │
│   ├── Производительность: 250,000 тайлов на 1 км²                           │
│   ├── Боёвка: точное позиционирование, 1 шаг = 0.5 тайла                    │
│   ├── Навигация: разумный путь поиска                                       │
│   └── Совместимость: игрок = 1 тайл, дракон = 4+ тайлов                     │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 2. Структура Мировой Карты

### 2.1 Секторы — Базовые единицы карты мира

**Определение:** Сектор — это минимальная единица навигации на карте мира (10×10 км).

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           СТРУКТУРА СЕКТОРА                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   Сектор (10×10 км) — ячейка на карте мира                                  │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │                                                                      │   │
│   │   Содержит 1-10 локаций (точек интереса):                           │   │
│   │                                                                      │   │
│   │   ┌────────────┐                                                    │   │
│   │   │ 🏘️ Город   │  1×1 км = 250,000 тайлов                          │   │
│   │   └────────────┘                                                    │   │
│   │         ┌──────┐                                                    │   │
│   │         │ 🌲   │  Дикие земли (не загружаются как сцена)            │   │
│   │         └──────┘                                                    │   │
│   │   ┌──────┐                                                          │   │
│   │   │ 🏛️   │  Храм 200×200 м = 10,000 тайлов                         │   │
│   │   └──────┘                                                          │   │
│   │                                                                      │   │
│   │   Тип местности: forest                                              │   │
│   │   Ци плотность: 120                                                  │   │
│   │   Опасность: 3                                                       │   │
│   │   Точки интереса: 3 (город, храм, деревня)                           │   │
│   │                                                                      │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                              │
│   ТАЙЛЫ НЕ ХРАНЯТСЯ на уровне сектора!                                      │
│   Только метаданные и список локаций.                                       │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

**Характеристики сектора:**

| Параметр | Тип | Описание |
|----------|-----|----------|
| `sectorId` | string | Уникальный идентификатор (X_Y) |
| `terrainType` | enum | Основной тип местности |
| `elevation` | float | Высота над уровнем моря (0-10000) |
| `qiDensity` | int | Плотность Ци в секторе |
| `dangerLevel` | int | Уровень опасности (1-9) |
| `climate` | enum | Климатическая зона |
| `discovered` | bool | Открыт ли сектор |
| `lastVisited` | long | Timestamp последнего посещения |
| `locations` | List | Список локаций в секторе |

```
│                                                                              │
│   1×1 (один тайл = 2×2 м):                                                  │
│   ├── Игрок, NPC                                                            │

│   ├── Куст, камень                                                          │
│   └── Сундук, дверь                                                         │
│                                                                              │
│   2×1 (два тайла горизонтально = 4×2 м):                                    │
│   ├── Лошадь, повозка                                                       │
│   └── Узкий стол                                                            │
│                                                                              │
│   1×2 (два тайла вертикально = 2×4 м):                                      │
│   ├── Столб, флагшток                                                       │
│   └── Высокая статуя                                                        │
│                                                                              │
│   2×2 (четыре тайла = 4×4 м):                                               │
│   ├── Небольшой дом                                                         │
│   ├── Большой дракон                                                        │
│   └── Алтарь                                                                │

│   ├── Храм, дворец                                                          │

```

### 6.2 Структура данных многоячеечного объекта

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                      СТРУКТУРА МНОГОЯЧЕЕЧНОГО ОБЪЕКТА                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   class MultiTileObject {                                                   │
│       // === ИДЕНТИФИКАЦИЯ ===                                              │
│       string objectId;           // Уникальный ID                           │
│       string objectTypeId;       // Тип объекта (tree, house, dragon...)    │
│                                                                              │
│       // === РАЗМЕР ===                                                      │
│       int width;                // Ширина в тайлах (1, 2, 3...)             │
│       int height;               // Высота в тайлах (1, 2, 3...)             │
│       Vector2Int origin;        // Левый верхний угол (якорь)               │
│                                                                              │
│       // === ЗАНЯТЫЕ ТАЙЛЫ ===                                               │
│       List<Vector2Int> occupiedTiles;  // Все занятые тайлы                 │
│                                         // Рассчитывается автоматически     │
│                                                                              │
│       // === ПАРАМЕТРЫ ===                                                   │
│       bool isPassable;          // Можно ли пройти                          │
│       float moveCost;           // Множитель движения                       │
│       bool blocksVision;        // Блокирует видимость                      │
│       int coverBonus;           // Бонус укрытия                            │
│                                                                              │
│       // === МЕТОДЫ ===                                                      │
│       List<Vector2Int> GetOccupiedTiles() {                                 │
│           // Возвращает все тайлы, занимаемые объектом                      │
│           // origin.x ... origin.x + width - 1                              │
│           // origin.y ... origin.y + height - 1                             │
│       }                                                                      │
│                                                                              │
│       bool IsTileOccupied(Vector2Int tile) {                                │
│           // Проверяет, занят ли конкретный тайл                            │
│       }                                                                      │
│   }                                                                          │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```
