## 📋 Обзор

Документ сравнивает две модели системы прорыва и рассчитывает игровое время, необходимое для прохождения с L1 на L9 при разных методах восполнения Ци.

---

## 🔷 Модель А: Растущий резервуар

### Принцип
- **Ёмкость ядра растёт** с каждым уровнем
- **Требуемое количество Ци для прорыва** = coreCapacity × multiplier
- Плотность Ци — отдельный параметр качества (влияет на урон техник)

### Формула ёмкости
```
coreCapacity = 1000 × 1.1^totalSubLevels
```

### Требования для прорыва
- **Малый уровень:** coreCapacity × 10 Ци
- **Большой уровень:** coreCapacity × 100 Ци

---

## 🔶 Модель Б: Сжатие в постоянном объёме

### Принцип
- **Ёмкость ядра фиксирована** (например, 1000 единиц объёма)
- **Плотность Ци растёт** с уровнем: qiDensity = 2^(level-1)
- Прорыв требует заполнения ядра Ци текущего уровня плотности

### Формула "количества Ци" для прорыва
```
effectiveQiRequired = baseQi × qiDensity
```

### Интерпретация
- Ядро вмещает фиксированный объём, но Ци становится "гуще"
- Для накопления 1000 единиц Ци плотности 4 (L3) нужно поглотить в 4 раза больше "сырой Ци" из окружения

## 📈 Вариант 2: Медитация 8 часов/день в стартовой локации

### Условия
- Микроядро: 10% coreCapacity/сутки (24 часа)
- Медитация: 8 часов/день = 28800 секунд
- Проводимость для поглощения из окружения
- environmentMult = 0.5 (стартовая локация, низкая плотность Ци)
- После прорыва currentQi = 0

### Формула поглощения
```
qiFromMeditation = meditationTime × conductivity × environmentMult
qiFromMicroCore = coreCapacity × 0.1 × (meditationHours / 24)
```

### Расчёт для Модели А

**L1→L2:**
- coreCapacity = 1,000
- Требуется: 100,000 Ци
- Проводимость: 1000/360 = 2.78 ед/сек
- Медитация: 28800 × 2.78 × 0.5 = 40,000 Ци/день
- Микроядро за 8 часов: 1000 × 0.1 × (8/24) = 33 Ци
- Итого в день: 40,033 Ци
- Дней: 100,000 / 40,033 = **2.5 дня**

**L2→L3:**
- coreCapacity = 2,594
- Требуется: 259,400 Ци
- Проводимость: 2594/360 = 7.21 ед/сек
- Медитация: 28800 × 7.21 × 0.5 = 103,824 Ци/день
- Микроядро: 2594 × 0.1 × (8/24) = 86 Ци
- Итого: 103,910 Ци/день
- Дней: 259,400 / 103,910 = **2.5 дня**

**Паттерн:** Время остаётся постоянным (~2.5 дня), так как и требуемое Ци, и проводимость растут пропорционально!

| Прорыв | Требуется Ци | Проводимость | Ци/день | Дней |
|--------|--------------|--------------|---------|------|

**Итого Модель А:** 20 дней = **~3 недели**

## 📈 Вариант 4: Зарядник вместо камней Ци

### Условия
- Зарядник — ограничитель скорости передачи Ци
- Проводимость зарядника НЕ зависит от практика
- Практик должен ПОДБИРАТЬ зарядник под свою проводимость

### Таблица зарядников

| Refined | 10 ед/сек | L2-L3 |
| Perfect | 25 ед/сек | L4-L6 |
| Transcendent | 50 ед/сек | L7-L9 |

### Формула скорости
```
effectiveRate = min(charger.conductivity, practitioner.conductivity)
```

### Проблема: Узкое место

**Для Модели А:**
- L5 практик: проводимость 125.7 ед/сек
- Perfect зарядник: 25 ед/сек
- **Узкое место — зарядник!** Скорость = 25 ед/сек

**Для Модели Б:**
- Проводимость практика фиксирована (1000/360 = 2.78 ед/сек)
- Common зарядник (5 ед/сек) — практика — узкое место
- Но при росте плотности Ци зарядник становится узким местом!

## 🎯 Анализ и выводы

### Модель А: Растущий резервуар

**Плюсы:**
- Чувство прогресса (ёмкость растёт)
- Баланс с зарядниками — нужен апгрейд снаряжения
- Больше игрового времени = больше контента

**Минусы:**
- Очень долгое время только с микроядром
- Зарядник становится узким местом на L5+
- Требует ~130 Кристаллов для быстрого прогресса

**Геймдизайн:** Подходит для игры с акцентом на экономику ресурсов и развитие инфраструктуры.

### Модель Б: Сжатие в постоянном объёме

**Плюсы:**
- Быстрый прогресс
- Common зарядник достаточен всегда
- Меньше расход камней Ци

**Минусы:**
- Меньше игрового времени
- Нет ощущения роста ёмкости
- Может казаться "слишком легко"

**Геймдизайн:** Подходит для игры с акцентом на боевку и исследование, а не на культивацию.

## 🔧 Рекомендации

### Для баланса Модели А с зарядниками

1. **Увеличить проводимость зарядников:**
   - Common: 10 ед/сек (вместо 5)
   - Refined: 25 ед/сек (вместо 10)
   - Perfect: 50 ед/сек (вместо 25)
   - Transcendent: 100 ед/сек (вместо 50)

2. **Или уменьшить проводимость практика:**
   - conductivity = coreVolume / 720 (вместо 360)
   - Это сделает зарядники менее лимитирующими

3. **Или добавить перки на проводимость:**
   - Позволит практику "обогнать" зарядник при развитии

### Для баланса Модели Б

1. **Добавить затраты на прорыв:**
   - Специальные ресурсы (травы, минералы)
   - Риск неудачи (травма при провале)
   - Время самого прорыва (не мгновенно)

2. **Ограничить доступ к медитации:**
   - Нужны специальные локации
   - Нужны техники медитации
   - Конкуренция за ресурсы Ци
