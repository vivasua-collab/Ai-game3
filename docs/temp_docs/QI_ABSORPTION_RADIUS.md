## 📋 Обзор

Документ разрабатывает механику радиуса сферы поглощения Ци практиком в зависимости от уровня культивации.

---

## 🔷 Принципы из ЛОРА

### Базовые правила

1. **Практик собирает ВСЮ Ци в зоне радиуса R**
2. **Ци в зоне ИСЧЕЗАЕТ** (ρ_окр → 0)
3. **Восстановление из недр — очень медленное** (G = 10 ед/ч·м² минимум)
4. **Скорость поглощения = проводимость практика** (ед/сек)

### Из лора

```
"радиус поля зрения, по умолчанию - 10 м"
```

```
"практики с повышением уровня могут взаимодействовать 
с Ци в большем радиусе и 'пылесосить' её"
```

## 🎯 Рекомендуемый вариант: №4

### Формула

```
R = R_base × (1 + 0.2 × (2^(level/3) - 1))
```

Где:
- `R_base = 10` метров (базовый радиус)
- `level` — большой уровень культивации (1-9)

### Обоснование

1. **Плавный рост на низких уровнях** — L1→L4: радиус 10→13м (+30%)
2. **Ускорение на средних** — L4→L6: радиус 13→16м (+23%)
3. **Заметный рост на высоких** — L6→L9: радиус 16→22м (+38%)
4. **Общий рост объёма** — ×10.6 (сбалансировано для Модели В)

### Лорное обоснование

```
L1-L3: "Пробуждённое Ядро" → "Пламя Внутреннего Огня"
       Практик только учится чувствовать Ци вокруг.
       Радиус мал, поглощение локальное.

L4-L6: "Объединение Тела и Духа" → "Разрыв Пелены"
       Ядро начинает взаимодействовать с внешними потоками Ци.
       Радиус растёт, практик "слышит" Ци ландшафта.

L7-L9: "Вечное Кольцо" → "Бессмертное Ядро"
       Ядро сливается с миром, практик становится частью поля Ци.
       Радиус максимален, возможно пассивное поглощение даже во сне.
```

## 🔄 Влияние на баланс прорывов (Модель В)

### Пересчёт Варианта 2 с учётом радиуса

**Условия:**
- Внешнее кольцо (ρ = 150 ед/м³)
- Медитация 8 часов/день
- Запас Ци в зоне = лимитирующий фактор

| Прорыв | Требуется Ци | Запас в зоне (L) | Запас в зоне (L-1) | Дней |

*Нужно несколько "циклов осушения-восстановления" зоны.

**Формула циклов:**
```
cycles = ceil(Требуется / Запас_в_зоне)
дней = cycles × (время_поглощения + время_восстановления)
```

### Время восстановления зоны

Из лора:
```
Минимальная скорость истечения Ци G = 10 ед/ч·м²
```

**Время восстановления зоны до 90%:**
```
t_recover = 0.9 × V_zone × ρ_окр / (G × S_zone)
где S_zone = 4 × π × R² (площадь поверхности сферы)
```

| Уровень | R (м) | S_zone (м²) | Время восстановления |
|---------|-------|-------------|---------------------|
| L1 | 10 | 1,257 | ~450 часов (~19 дней) |
| L5 | 15 | 2,827 | ~380 часов (~16 дней) |
| L9 | 22 | 6,082 | ~350 часов (~15 дней) |

## 🎯 Итоговая формула для Модели В

### Полный расчёт времени прорыва

```
1. Определить радиус: R = 10 × (1 + 0.2 × (2^(level/3) - 1))
2. Рассчитать объём: V = 4/3 × π × R³
3. Запас Ци в зоне: Qi_zone = ρ_окр × V
4. Скорость поглощения: rate = проводимость практики
5. Время поглощения: t_absorb = Qi_zone / rate
6. Время восстановления: t_recover ~ 15-20 дней
7. Циклы: cycles = ceil(Требуется / Qi_zone)
8. Общее время: t_total = cycles × (t_absorb + t_recover)
```

### Итоговая таблица L1→L9 (Внешнее кольцо)

| Прорыв | Требуется | Qi_zone | Циклов | Дней (с восстановлением) |
|--------|-----------|---------|--------|--------------------------|
| L1→L2 | 5,188 | 628,350 | 1 | < 1 |
| L2→L3 | 26,908 | 806,550 | 1 | < 1 |
| L3→L4 | 139,600 | 1,085,700 | 1 | < 1 |
| L4→L5 | 724,160 | 1,380,450 | 1 | 16.5 |
| L5→L6 | 3,756,480 | 2,120,550 | 2 | 32.7 |
| L6→L7 | 19,486,720 | 2,573,550 | 8 | 131.2 |
| L7→L8 | 101,088,000 | 3,664,350 | 28 | 459.2 |

**Итого:** ~2,363 дней = **6.5 лет** (вместо 2.8 года без учёта восстановления!)
