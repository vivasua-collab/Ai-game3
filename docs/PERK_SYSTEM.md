## 1. Обзор Системы

### 1.1 Что такое Перки?

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                      ЧТО ТАКОЕ ПЕРКИ?                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   Перк (Perk/Feature/Trait) — постоянная пассивная способность персонажа.   │
│                                                                              │
│   Отличия от других систем:                                                 │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │                        │                    │                         │ │
│   │   ПЕРКИ                │   БАФФЫ            │   НАВЫКИ                │ │
│   ├───────────────────────────────────────────────────────────────────────┤ │
│   │ Длительность:          │                    │                         │ │
│   │   Постоянные           │   Временные        │   Развиваются           │ │
│   │   (до снятия)          │   (секунды/минуты) │   (тренировка)          │ │
│   ├───────────────────────────────────────────────────────────────────────┤ │
│   │ Получение:             │                    │                         │ │
│   │   События, выбор,      │   Техники,         │   Повторяющиеся         │ │
│   │   достижения, рождение │   предметы         │   действия              │ │
│   ├───────────────────────────────────────────────────────────────────────┤ │
│   │ Эффект:                │                    │                         │ │
│   │   Уникальные           │   Стандартные      │   Прогрессивные         │ │
│   │   особенности          │   бонусы           │   улучшения             │ │
│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│   Перк = "Твоя уникальная особенность"                                      │
│   Бафф = "Временное усиление"                                               │
│   Навык = "Развитая способность"                                            │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 1.2 Ключевые принципы

| Принцип | Описание |
|---------|----------|
| **Перманентность** | Перк действует постоянно, пока не удалён |
| **Уникальность** | Каждый перк — особенный эффект, не просто +X% |
| **Ограниченность** | Ограниченное количество слотов для перков |
| **Вес перка** | Сильные перки занимают больше слотов |
| **Получение через игру** | Зарабатываются действиями, событиями, выборами |

## 2. Типы Перков

### 2.1 Категории по происхождению

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                      КАТЕГОРИИ ПЕРКОВ                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   🌟 ВРОЖДЁННЫЕ (Innate)                                                    │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │ • Получаются при создании персонажа или пробуждении                   │ │
│   │ • Нельзя удалить или изменить                                         │ │
│   │ • Определяют "судьбу" персонажа                                       │ │
│   │                                                                        │ │
│   │ Примеры:                                                               │ │
│   │ • "Дар Небес" — +50% к регенерации Ци                                 │ │
│   │ • "Проклятая кровь" — -20% HP, +30% урон тьмой                        │ │
│   │ • "Небесное тело" — иммунитет к болезням                              │ │
│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│   🎯 ПРИОБРЕТЁННЫЕ (Acquired)                                               │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │ • Получаются через игру: события, выборы, достижения                  │ │
│   │ • Можно заменить (с потерей)                                          │ │
│   │ • Зависят от действий игрока                                          │ │
│   │                                                                        │ │
│   │ Примеры:                                                               │ │
│   │ • "Закалка тела" — +15% к VIT после 100 часов тренировок              │ │
│   │ • "Убийца драконов" — +50% урона драконам                             │ │
│   │ • "Выживший" — 1 раз выживает при смертельном уроне                   │ │
│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│   ⚜️ КУЛЬТИВАЦИОННЫЕ (Cultivation)                                          │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │ • Получаются при прорывах уровней культивации                         │ │
│   │ • Зависят от пути культивации                                         │ │
│   │ • Растут в силе с уровнем                                             │ │
│   │                                                                        │ │
│   │ Примеры:                                                               │ │
│   │ • "Душа меча" (L3) — техники меча требуют -15% Ци                     │ │
│   │ • "Огненное ядро" (L5) — +25% урон огнём, иммунитет к холоду          │ │
│   │ • "Небесное дыхание" (L7) — полёт                                      │ │
│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│   💀 ПРОКЛЯТЫЕ (Cursed)                                                     │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │ • Навязанные извне (ритуалы, враги, артефакты)                        │ │
│   │ • Имеют и плюсы, и минусы                                             │ │
│   │ • Сложно снять                                                        │ │
│   │                                                                        │ │
│   │ Примеры:                                                               │ │
│   │ • "Кровавый долг" — +40% урон, -5% HP за технику                      │ │
│   │ • "Проклятие молчания" — +50% ментальная защита, нельзя говорить     │ │
│   │ • "Паразит Ци" — -30% регенерация Ци, +20% вампиризм                  │ │
│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 2.2 Категории по эффекту

| Категория | Описание | Примеры |
|-----------|----------|---------|
| **Боевые** | Влияют на бой | +урон, +крит, новые атаки |
| **Выживание** | Защита и регенерация | HP, сопротивления, иммунитеты |
| **Культивация** | Ци и развитие | Регенерация Ци, прорывы |
| **Социальные** | Взаимодействие с NPC | Торговля, репутация, диалоги |
| **Исследование** | Мир и путешествия | Скорость, видение, поиск |
| **Крафт** | Создание предметов | Качество, скорость, рецепты |
| **Мета** | Влияют на саму игру | Удача, опыт, второй шанс |

```
│                      ВЕС ПЕРКА                                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │

│   │                                                                        │ │
│   │   ┌─────┐          Вес: 1                                              │ │
│   │   │  ⭐ │          Слабый перк (малый бонус)                           │ │
│   │   └─────┘          Пример: "+5% к скорости атаки"                      │ │

│   │   ┌─────┬─────┬─────┐ Вес: 3                                           │ │
│   │   │  ⭐⭐⭐ │     │   Мощный перк                                       │ │
│   │   └─────┴─────┴─────┘ Пример: "Второй шанс при смерти (1 раз/день)"    │ │
│   │                                                                        │ │
│   │   ┌─────┬─────┬─────┬─────┐ Вес: 4                                     │ │
│   │   │  ⭐⭐⭐⭐│         │ Легендарный перк                                │ │

│   │                                                                        │ │

│                                                                              │
│   Правило: чем сильнее эффект, тем больше вес                               │
│                                                                              │
│   Пример заполнения 5 слотов:                                               │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │   [Вес 2] [Вес 2] [Вес 1] = 5 слотов занято                           │ │
│   │   или                                                                   │ │
│   │   [Вес 3] [Вес 1] [Вес 1] = 5 слотов занято                           │ │
│   │   или                                                                   │ │
│   │   [Вес 1] [Вес 1] [Вес 1] [Вес 1] [Вес 1] = 5 слотов                  │ │
│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 4. Механики Эффектов

### 4.1 Типы эффектов перков

```

│                      ТИПЫ ЭФФЕКТОВ ПЕРКОВ                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   1. ПАССИВНЫЕ БОНУСЫ (Passive Bonuses)                                     │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │ Постоянно активные модификаторы.                                       │ │
│   │                                                                        │ │
│   │ • "Железная воля" — +25% к ментальной защите                          │ │
│   │ • "Быстрые рефлексы" — +15% к уклонению                               │ │
│   │ • "Крепкое тело" — +10% к макс. HP                                     │ │
│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│   2. ТРИГГЕРНЫЕ ЭФФЕКТЫ (Triggered Effects)                                 │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │ Активируются при определённых условиях.                                │ │
│   │                                                                        │ │
│   │ • "Адреналин" — при HP < 30%: +50% скорость атаки на 10 сек           │ │
│   │ • "Месть" — при получении урона: 20% шанс нанести 50% урона врагу     │ │
│   │ • "Второе дыхание" — 1 раз в бою: при HP = 0, восстановить 30% HP     │ │
│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│   3. УСЛОВНЫЕ БОНУСЫ (Conditional Bonuses)                                  │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │ Активны только при выполнении условия.                                 │ │
│   │                                                                        │ │
│   │ • "Ночной охотник" — ночью: +30% урон, +20% скорость                  │ │
│   │ • "Горный дух" — в горах: +40% защита, иммунитет к высоте             │ │
│   │ • "Кровавый путь" — если враг кровоточит: +50% урон ему               │ │
│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│   4. МОДИФИКАТОРЫ ДЕЙСТВИЙ (Action Modifiers)                               │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │ Изменяют свойства действий.                                            │ │
│   │                                                                        │ │
│   │ • "Пронзающий удар" — атаки пробивают 20% брони                       │ │
│   │ • "Быстрый каст" — время каста техник -20%                            │ │
│   │ • "Экономный" — расход Ци техник -15%                                 │ │
│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│   5. НОВЫЕ СПОСОБНОСТИ (New Abilities)                                      │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │ Дают совершенно новые возможности.                                     │ │
│   │                                                                        │ │
│   │ • "Ночное зрение" — видит в темноте                                    │ │
│   │ • "Полёт" — может летать (требует L7+)                                 │ │
│   │ • "Телепатия" — может общаться мысленно на 100м                        │ │

│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│   6. МЕТРИЧЕСКИЕ ЭФФЕКТЫ (Metric Effects)                                   │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │ Влияют на игровые метрики.                                             │ │
│   │                                                                        │ │
│   │ • "Удачливый" — +20% шанс редкого лута                                 │ │
│   │ • "Быстрый ученик" — +25% получаемого опыта                           │ │
│   │ • "Торговец" — цены покупки -15%, продажи +15%                        │ │
│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 6. Интеграция с Существующими Системами

### 6.1 Связь с BUFF_SYSTEM

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                      ПЕРКИ vs БАФФЫ                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   ПЕРКИ влияют на базовые параметры, к которым потом применяются баффы:    │
│                                                                              │
│   Пример расчёта урона:                                                    │
│   ┌───────────────────────────────────────────────────────────────────────┐ │
│   │                                                                        │ │
│   │   База: 100 урона                                                      │ │
│   │                                                                        │ │
│   │   Перк "Берсерк" (пассивный): +20% урон                               │ │
│   │   → modified_base = 100 × 1.20 = 120                                  │ │
│   │                                                                        │ │
│   │   Перк "Ночной охотник" (условный, ночь): +30% урон                   │ │
│   │   → modified_base = 120 × 1.30 = 156                                  │ │
│   │                                                                        │ │
│   │   Бафф "Ярость" (временный, 30 сек): +25% урон                        │ │
│   │   → final = 156 × 1.25 = 195                                          │ │
│   │                                                                        │ │
│   │   Итого: 195 урона                                                     │ │
│   │                                                                        │ │
│   └───────────────────────────────────────────────────────────────────────┘ │
│                                                                              │
│   Порядок применения:                                                       │
│   1. Базовое значение                                                       │
│   2. Перк-модификаторы (постоянные)                                        │
│   3. Условные перки (если активны)                                         │
│   4. Баффы/дебаффы (временные)                                             │
│   5. Мягкие капы                                                            │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 6.2 Связь с MODIFIERS_SYSTEM

| Модификатор | Источник | Время действия |
|-------------|----------|----------------|
| База | Характеристики | Постоянно |
| Перк | Система перков | Постоянно |
| Бафф | Система баффов | Временно |
| Окружение | Мир | Пока в зоне |

```

│                                                                              │
│   class PerkEffect                                                         │

│   }                                                                         │
│                                                                              │
│   enum PerkEffectType                                                      │
│   {                                                                         │
│       PassiveBonus,       // Постоянный бонус                              │
│       TriggeredEffect,    // Срабатывает при условии                       │
│       ConditionalBonus,   // Активен при условии                           │

│       OnCrit,              // При критическом ударе                        │

│       OnRest,              // При отдыхе                                   │
```
