## 🔄 Машина состояний

### Переходы между состояниями

| Текущее | Условие | Следующее |
|---------|---------|-----------|
| idle | Игрок в радиусе агрессии | chase |
| idle | Точка патруля | patrol |
| patrol | Игрок в радиусе агрессии | chase |
| chase | Цель в радиусе атаки | attack |
| chase | Цель потеряна | patrol |
| attack | HP < 20% | flee |
| attack | Цель мертва | patrol |
| flee | В безопасности | patrol |
| flee | HP = 0 | dead |

## ⚡ Spinal AI (Рефлексы)

### Концепция

Быстрые автоматические реакции без сознательного решения.

Spinal AI — это "подсознание" NPC, работающее без участия сложных алгоритмов.

### Трёхуровневая модель нервной системы

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    ИЕРАРХИЯ НЕРВНОЙ СИСТЕМЫ NPC                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   УРОВЕНЬ 3: BRAIN CONTROLLER (Мозг)                                        │
│   ├── Сложные решения, диалоги, планирование                               │
│   ├── Скорость: 100-500мс (медленно)                                       │
│   └── Затраты: Высокие (LLM или продвинутый AI)                            │
│                                                                              │
│   УРОВЕНЬ 2: NEURAL ROUTER (Нервная система)                                │
│   ├── Маршрутизация сигналов, буферизация                                  │
│   ├── Скорость: 10-50мс                                                    │
│   └── Затраты: Минимальные                                                 │
│                                                                              │
│   УРОВЕНЬ 1: SPINAL AI (Позвоночник) ← ПЕРВИЧНЫЙ                           │
│   ├── Рефлексы, автоматические реакции                                     │
│   ├── Скорость: 1-10мс (очень быстро)                                      │
│   └── Затраты: Минимальные                                                 │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Типы входных сигналов

| Тип сигнала | Описание | Пример |
|-------------|----------|--------|
| collision | Столкновение | Столкновение с препятствием |
| damage | Получен урон | Удар противника |
| danger_nearby | Опасность рядом | Приближение врага |
| edge_detected | Край обнаружен | Обрыв, вода |
| loud_sound | Громкий звук | Взрыв, крик |
| temperature | Температура | Огонь, холод |
| balance_lost | Потеря равновесия | Подножка |
| hunger | Голод | Низкий уровень сытости |
| fatigue | Усталость | Низкий уровень энергии |

### Типы рефлексов

| Рефлекс | Триггер | Действие |
|---------|---------|----------|
| Dodge | Снаряд приближается | Уклонение |
| Flinch | Получен урон | Отшатывание |
| Flee | HP < порог | Бегство |
| QiShield | Атака Ци | Авто-щит |
| Alert | Движение рядом | Настороженность |
| StepBack | Край обнаружен | Отойти от края |
| TurnToSound | Громкий звук | Повернуться |
| BalanceRecover | Потеря равновесия | Восстановить баланс |

### Приоритеты рефлексов

| Приоритет | Рефлекс | Описание |
|-----------|---------|----------|
| 100 (критический) | danger_dodge | Уклонение от критической опасности |
| 95 | qi_shield_reflex | Авто-щит при атаке Ци |
| 90 | pain_flinch | Вздрогнуть от боли |
| 85 | edge_retreat | Отойти от края |
| 80 | collision_stumble | Пошатнуться при столкновении |
| 70 | balance_recover | Восстановить равновесие |
| 30 | sound_orient | Повернуться на звук |

### Параметры рефлексов

| Параметр | Описание |
|----------|----------|
| triggerDistance | Расстояние срабатывания |
| reactionTime | Время реакции (мс) |
| cooldown | Перезарядка (мс) |
| qiCost | Стоимость Ци (для qi_shield) |
| priority | Приоритет (0-100) |
| canInterrupt | Можно ли прервать другой рефлекс |

## 🔀 Neural Router (Нервная система)

### Назначение

Neural Router — промежуточный уровень между Spinal AI и Brain Controller.
Обеспечивает маршрутизацию, буферизацию и приоритизацию сигналов.

### Функции Neural Router

| Функция | Описание |
|---------|----------|
| Буферизация | Очереди сигналов с приоритетами |
| Маршрутизация | Spinal ↔ Brain |
| Агрегация | Объединение похожих сигналов |
| Фильтрация | Отсечение шума |
| Кэширование | Кэш частых запросов |
| Внимание | Управление фокусом NPC |

### Приоритеты сигналов

| Приоритет | Тип сигнала | Пример |
|-----------|-------------|--------|
| critical | Опасность для жизни | Атака L9 практика |
| high | Важные события | Появление врага |
| medium | Обычные события | Звук шагов |
| low | Фоновые сигналы | Погода, время суток |

## 🧠 Brain Controller (Мозг)

### Назначение

Brain Controller — когнитивный уровень AI для сложных решений, диалогов и планирования.

### Типы запросов к Brain

| Тип запроса | Описание |
|-------------|----------|
| dialogue_generate | Генерация реплики в диалоге |
| decision_make | Принятие сложного решения |
| plan_create | Создание долгосрочного плана |
| situation_analyze | Анализ ситуации |
| emotion_determine | Определение эмоции |
| goal_evaluate | Оценка целей |

### Контекст для Brain

| Данные | Описание |
|--------|----------|
| npc.identity | Идентичность (имя, роль, личность) |
| npc.state | Текущее состояние (HP, Qi, настроение) |
| situation.location | Локация, время суток |
| situation.nearby | Существа поблизости |
| player.info | Информация об игроке |
| player.history | История взаимодействий |
| world.events | Глобальные события |

## 🧠 Behaviour Tree

### Узлы дерева

| Тип | Описание |
|-----|----------|
| Selector | Выбирает первую успешную ветку |
| Sequence | Выполняет все узлы по порядку |
| Condition | Проверка условия |
| Action | Выполнение действия |

### Пример дерева

```
Selector
├── Sequence (Combat)
│   ├── Condition: InCombat?
│   ├── Condition: HasTarget?
│   └── Action: Attack
│
├── Sequence (Alert)
│   ├── Condition: IsAlert?
│   └── Action: SearchArea
│
├── Sequence (Patrol)
│   ├── Condition: HasPatrolRoute?
│   └── Action: Patrol
│
└── Action: Idle
```

## 🎯 Целевая система

### Выбор цели

1. **Проверка aggroRadius**
2. **Оценка угрозы** (уровень, поведение)
3. **Приоритеты** (класс, раса, фракция)
4. **Выбор основной цели**

### Переключение цели

| Условие | Действие |
|---------|----------|
| Текущая цель мертва | Выбрать новую |
| Более опасная цель | Переключиться |
| HP < 20% | Бегство |

## 💬 Социальное взаимодействие

### Расчёт отношения (Attitude + PersonalityTrait)

> **Изменено в Fix-07:** Disposition заменён на `Attitude` (числовое -100..+100) + `PersonalityTrait [Flags]`. DATA_MODELS.md обновлён соответственно.

```
finalAttitude = 
  + personalAttitude
  + sectRelation × 0.5
  + factionRelation × 0.3
  + nationRelation × 0.2
  + attireModifier
  + reputationModifier
```

### Пороги Attitude

| Значение | Отношение | Поведение |
|----------|-----------|-----------|
| 80-100 | Клятвенный союзник | Самопожертвование |
| 50-79 | Союзник | Лояльность |
| 10-49 | Друг | Помощь, торговля |
| -9-9 | Нейтральный | Безразличие |
| -20--10 | Недоброжелательный | Избегание |
| -50--21 | Враждебный | Атака если спровоцирован |
| -100--51 | Ненависть | Атака без предупреждения |

### PersonalityTrait [Flags]

| Флаг | Значение | Описание |
|------|----------|----------|
| Aggressive | 1 | Склонность к агрессии |
| Cautious | 2 | Осторожность, избегание риска |
| Treacherous | 4 | Предательство при возможности |
| Ambitious | 8 | Стремление к власти |
| Loyal | 16 | Верность союзникам |
| Pacifist | 32 | Избегание конфликта |
| Curious | 64 | Любопытство, исследование |
| Vengeful | 128 | Мстительность |
