## 📋 Обзор

Ци — основная энергия мира культивации. Система Ци управляет накоплением, расходованием и использованием этой энергии.

---

## 🌟 Основные принципы

### Свойства Ци

1. **Квантованность** — Ци неделима, минимальная единица = 1
2. **Два состояния:** Спокойная и Хаотичная
3. **Агрегатные состояния:**
   - Газ — в окружающей среде
   - Твердое тело — в кристаллах (камни духа)
   - Жидкость — в теле практиков
   - Плазма — в техниках и массивах

### Плотность Ци

> **Источник истины:** [ALGORITHMS.md](./ALGORITHMS.md) §3.3 "Плотность Ци"

**Формула:** `Плотность Ци = 2^(уровень культивации - 1)`

Качество Ци геометрически растёт ×2 за каждый уровень. Подробная таблица — в ALGORITHMS.md.

## 💖 Ядро практика

### Ёмкость ядра

**Формула:**
```
coreCapacity = baseCapacity × 1.1^totalSubLevels
```

Где:
- `baseCapacity` = 1000 (L1.0)
- `totalSubLevels` = (level - 1) × 10 + subLevel

### Таблица ёмкости

| Уровень | totalSubLevels | coreCapacity |
|---------|----------------|--------------|
| L1.0 | 0 | 1,000 |
| L2.0 | 10 | 2,594 |
| L3.0 | 20 | 6,727 |
| L4.0 | 30 | 17,450 |
| L5.0 | 40 | 45,260 |
| L6.0 | 50 | 117,390 |
| L7.0 | 60 | 304,480 |
| L8.0 | 70 | 789,750 |

## 🔌 Система меридиан

### Структура

1. **Ядро** — корень системы меридиан
2. **Основной ствол** — определяет проводимость
3. **Ветви** — по телу практика
4. **Узлы вывода** — точки выхода Ци

### Проводимость

**Базовая формула при становлении практиком:**
```
conductivity = coreVolume / 360 секунд
```

**Формула с бонусами от перков:**
```
finalConductivity = baseConductivity × (1 + conductivityBonus)
```

**Влияние:**
- Скорость поглощения Ци из окружающей среды
- Скорость вывода Ци для техник
- Ёмкость рабочего буфера меридиан

### Развитие проводимости

> **Важно:** Проводимость МОЖНО развивать через перки и техники!

| Источник бонуса | Тип | Величина |
|-----------------|-----|----------|
| Врождённый перк "Золотое качество тела" | Постоянный | +30% |
| Приобретённый перк "Закалка меридиан" | Постоянный | +15% |
| Приобретённый перк "Небесные каналы" | Постоянный | +20% |
| Техника "Небесные каналы" (активная) | Временный | +50% |
| Формация "Круг поглощения" | Локальный | +25% |

**Пример расчёта:**
```
Базовая проводимость L3.0: 6727 / 360 = 18.7 Ци/сек

С врождённым перком "Золотое качество тела" (+30%):
finalConductivity = 18.7 × 1.30 = 24.3 Ци/сек

С дополнительным перком "Закалка меридиан" (+15%):
finalConductivity = 18.7 × 1.45 = 27.1 Ци/сек
```

### Ограничение поглощения

**Правило:** Поглощение Ци НЕ может превышать проводимость меридиан!

```
absorbedQi = min(
    meditationTime × finalConductivity × environmentMult,
    availableQiInEnvironment
)
```

Это означает, что перки на "+скорость поглощения" эффективны только в пределах проводимости.

### Буфер меридиан

**Формула:**
```
bufferVolume = finalConductivity × 5 секунд
```

Меридианы — каналы передачи, не хранилища. Объём в них — рабочий буфер для быстрого вывода.

**Важно:** Бонусы проводимости также увеличивают буфер!

## 📊 Камни Ци (Духовные Камни)

### Типы камней

| Тип | Ци | Редкость |
|-----|-----|----------|
| Осколок | 1,000 | common |
| Фрагмент | 10,000 | uncommon |
| Камень | 100,000 | rare |
| Кристалл | 1,000,000 | epic |
| Сердце | 10,000,000 | legendary |
| Ядро | 100,000,000 | mythic |

### Испарение камней

**Переменные:**
- `ρ_крист` — плотность кристалла = 1.024×10⁹ ед/м³
- `k` — коэффициент равновесия = 0.001
- `ρ_равн` — равновесная плотность = k × ρ_крист
- `ρ_окр` — текущая плотность Ци в помещении
- `S_крист` — площадь поверхности кристалла
- `α` — коэффициент испарения

**Формула испарения:**
```
Q_испар = α × S_крист × (ρ_равн - ρ_окр)
```

> **Источник истины:** [ALGORITHMS.md](./ALGORITHMS.md) §2 "Буфер Ци"
>
> Буфер Ци защищает практика от ЛЮБОГО урона (техники Ци и физический).
> Детальные формулы и примеры расчёта — в ALGORITHMS.md.

**Кратко:**
- Ци защищает практика даже во сне
- Техники Ци: 90% поглощение, 3:1 соотношение, 10% пробитие
- Физический урон: 80% поглощение, 5:1 соотношение, 20% пробитие
- Щитовая техника: 100% поглощение, 1:1 (Ци) или 2:1 (физика)

## 🧘 Медитация

### Назначение

Медитация позволяет поглощать Ци из окружающей среды.

### Формула поглощения

```
absorbedQi = meditationTime × conductivity × environmentMultiplier
```

### Факторы эффективности

| Фактор | Влияние |
|--------|---------|
| Проводимость меридиан | Скорость поглощения |
| Плотность Ци в локации | Множитель среды |
| Техники медитации | Бонус к скорости |
| Формации | Увеличение поглощения |
| Уровень культивации | Бонус от плотности ядра |

### Зарядники Ци

Специальное снаряжение для ускорения медитации:

| Грейд | Проводимость | Буфер | Слоты | Эффективность |
|-------|-------------|-------|-------|---------------|
| Damaged | ×0.5 | ×0.5 | +0 | -10% |
| Common | ×1.0 | ×1.0 | +0 | 0% |
| Refined | ×1.3 | ×1.5 | +1 | +5% |
| Perfect | ×1.7 | ×2.5 | +2 | +15% |
| Transcendent | ×2.5 | ×4.0 | +2 | +30% |

## 🔄 Прорыв уровня (Модель В — Растущий резервуар + Сжатие Ци)

> **Решение принято:** 2026-04-10 — Модель В выбрана как основная.
>
> Ключевое отличие от первоначальной модели: прорыв требует накопить Ци,
> равное capacity(nextLevel) × density(nextLevel), а после прорыва currentQi = 0.

### Условия

**Малый прорыв (под-уровень):**
- Накопить: `capacity(nextSubLevel) × qiDensity(nextSubLevel)` единиц Ци

**Большой прорыв (уровень):**
- Накопить: `capacity(nextLevel) × qiDensity(nextLevel)` единиц Ци

### Формулы Модели В

```
coreCapacity = 1000 × 1.1^totalSubLevels × qualityMultiplier
qiDensity = 2^(level-1)
effectiveQi = coreCapacity × qiDensity
conductivity = coreCapacity / 360

Требование прорыва = capacity(nextLevel) × density(nextLevel)
После прорыва: currentQi = 0 (ядро перестраивается)
```

### Процесс

1. Глубокая медитация
2. Концентрация Ци в ядре
3. Преобразование структуры ядра
4. Увеличение плотности Ци
5. Увеличение ёмкости ядра на ~10%

### Время

- Малый прорыв: ~480 игровых минут (8 часов)
- Большой прорыв: ~4800 игровых минут (80 часов)

## ⚙️ Расчёт эффективности техник

### Формула

```
эффективность = потрачено_Ци × качество_Ци
```

### Пример

- Техника требует 100 единиц Ци
- Практик 3-го уровня (плотность = 4)
- Эффективность: 100 × 4 = 400 базовых единиц

**Вывод:** Высококачественная Ци делает технику сильнее без увеличения объёма затрат.
