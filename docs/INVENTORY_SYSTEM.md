## ⚔️ Система экипировки

### Архитектура "Матрёшка"

Три независимых слоя:

1. **Базовый класс** (неизменен)
   - Тип экипировки
   - Уровень предмета
   - Материал
   - Требования

2. **Материал** (неизменен)
   - Тир материала
   - Базовые свойства
   - Особые эффекты

3. **Grade** (изменяем)
   - Качество: Damaged → Common → Refined → Perfect → Transcendent
   - Множители параметров
   - Дополнительные бонусы

## 🎯 Слоты экипировки

### Архитектура слотов (Гуманоид)

**Зонирование тела:**

| Зона тела | Armor Slot | Clothing Slot | Описание |
|-----------|------------|---------------|----------|
| **Голова** | `head_armor` | `head_clothing` | Шлем/корона + шапка/капюшон |
| **Торс** | `torso_armor` | `torso_clothing` | Нагрудник/кираса + рубашка/роба |
| **Ноги** | `legs_armor` | `legs_clothing` | Поножи + штаны |
| **Ступни** | `feet_armor` | `feet_clothing` | Сабатоны + сапоги/туфли |
| **Кисти** | `hands_armor` | `hands_clothing` | Боевые перчатки + обычные перчатки |

### Оружие

| Слот | Типы | Конфликты |
|------|------|-----------|
| `weapon_main` | Оружие, щит, инструмент | Блокируется `weapon_twohanded` |
| `weapon_off` | Оружие, щит, инструмент | Блокируется `weapon_twohanded` |
| `weapon_twohanded` | Двуручное оружие | Блокирует `weapon_main` + `weapon_off` |

### Зарядник

| Слот | Типы предметов | Лимит |
|------|----------------|-------|
| `charger` | Зарядник Ци (любой форм-фактор) | 1 |

> **Форм-факторы зарядников:** belt, bracelet, necklace, ring, backpack — все занимают слот `charger`.
> Форм-фактор влияет на характеристики (слоты камней, буфер, скорость), но не на слот экипировки.

### Аксессуары

| Слот | Типы предметов | Лимит |
|------|----------------|-------|
| `amulet` | Амулет, ожерелье, колье | 1 |
| `ring_left_1` | Кольцо | 1 |
| `ring_left_2` | Кольцо | 1 |
| `ring_right_1` | Кольцо | 1 |
| `ring_right_2` | Кольцо | 1 |

**Итого колец:** максимум 4 (по 2 на каждую руку)

### Правила экипировки

1. **Armor + Clothing** — раздельные слоты, не конфликтуют
   - Можно носить рубашку под броню
   - Можно носить штаны под поножи
   - Бонусы складываются

2. **Двуручное оружие** — занимает оба слота рук

3. **Зарядник** — один на персонажа (слот `charger`)

4. **Кольца** — по 2 на каждую руку, максимум 4

5. **Амулет** — только 1 слот

## 📈 Бонусы экипировки

### Типы бонусов

| Категория | Бонусы |
|-----------|--------|
| Combat | Урон, крит, пробитие |
| Defense | Броня, уклонение, HP |
| Qi | Регенерация, проводимость |
| Elemental | Сопротивления стихиям |
| Special | Вампиризм, удача |
| Utility | Скорость, скрытность |

### Структура бонуса

| Поле | Описание |
|------|----------|
| type | Тип бонуса |
| value | Значение |
| isPercent | % или абсолютный |
| source | Источник (base, grade, material, set) |

## 🎭 Сетовые бонусы

### Структура сета

| Поле | Описание |
|------|----------|
| id | ID сета |
| name | Название |
| pieces | ID предметов сета |
| setBonuses | Бонусы за количество |

### Пример

```
Fire Lord Set:
  Pieces: 5
  2 pieces: +20% сопротивление огню
  3 pieces: +5 интеллект, техника fire_aura
  5 pieces: +15 интеллект, +80% сопротивление огню
```
