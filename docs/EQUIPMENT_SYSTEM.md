## 🏗️ Архитектура "Матрёшка" (генерация)

> **Важно:** «Матрёшка» — это архитектура **ГЕНЕРАЦИИ** предметов (База × Грейд × Специализация), а **не** система слоёв экипировки.
> Слои экипировки (Матрёшка v1 — `layers: List<EquipmentLayer>`, несколько предметов в один слот) **упразднены** в EquipmentController v2.0.
> Принцип: **1 слот = 1 предмет** (решение 2026-04-29). См. `docs_temp/EQUIPPED_SPRITES_DRAFT.md` §6.

## 4️⃣ СИСТЕМА ПРОЧНОСТИ

### 4.1 Состояния

| Состояние | Прочность | Эффективность | Цвет |
|-----------|-----------|---------------|------|
| Pristine | 100% | 100% | 🟢 Зелёный |
| Good | 80-99% | 95% | 🟢 Светло-зелёный |
| Worn | 60-79% | 85% | 🟡 Жёлтый |
| Damaged | 20-59% | 60% | 🟠 Оранжевый |
| Broken | <20% | 20% | ⚫ Чёрный |

### 4.2 Потеря прочности

**Комбинированный метод:**

1. **Базовый износ** — при каждом использовании
2. **Ударный износ** — при сильных ударах (урон > порог)

**Порог удара** = твёрдость × 10

**Пример:**
- Железный меч (твёрдость 5): порог = 50 урона
- Если получен урон 80: дополнительный износ = (80-50)/10 = 3

### 4.3 Ремонт

| Метод | Требования | Результат |
|-------|------------|-----------|
| NPC-кузнец | Духовные камни + материалы | 100%, без риска |
| Самостоятельный | Навык + материалы | До 90%, риск понижения Grade |
| Ци-ремонт | Высокая культивация | Только для духовных материалов |

**Понижение Grade при плохом ремонте:**
- Качество < 50%: шанс понижения 25-40%
- Понижение теряет бонусы и эффекты

## 5️⃣.5 СИСТЕМА ЗАЧАРОВАНИЯ (Enchant System)

> **Источник истины:** EQUIPMENT_SYSTEM.md §5.5

### 5.5.1 Обзор

Зачарование — наложение магического эффекта на предмет экипировки. Это **5-й источник бонусов** (base, grade, material, set, **enchant**).

Зачарования добавляют бонусы поверх базовых параметров предмета. В отличие от грейда (изменяет множитель) и материала (определяет базу), зачарование — **независимый модификатор**.

### 5.5.2 Типы зачарований

| Категория | Примеры | Эффект | Сила |
|-----------|---------|--------|------|
| **Боевые** | Пылающий клинок, Громовой удар | +X% урона стихией | +5% ~ +50% |
| **Защитные** | Каменная кожа, Ледяной щит | +X% защиты / сопротивления | +5% ~ +30% |
| **Ци-зачарования** | Поток Ци, Резонанс ядра | +X% qiRestoration, -X% qi_cost | +5% ~ +25% |
| **Вампиризм** | Кровавый клинок, Душепийца | life_steal, thorns | +5% ~ +30% |
| **Специальные** | Удача, Зоркость | +X% luck, perception | +5% ~ +20% |

### 5.5.3 Структура зачарования

| Поле | Тип | Описание |
|------|-----|----------|
| id | string | Уникальный ID зачарования |
| name | string | Название |
| element | Element | Привязка к стихии (может быть neutral) |
| bonusType | string | Тип бонуса (damage, armor, life_steal...) |
| bonusValue | float | Базовая сила (+5%, +10%, +15%...) |
| isPercent | bool | % или абсолютный |
| tier | int | Тир зачарования (1-5) |
| maxStacks | int | Максимум стеков (обычно 1) |
| requirements | JSON | Требования к предмету (мин. грейд, мин. тир материала) |

### 5.5.4 Тиры зачарований

| Тир | Сила бонуса | Требование к предмету | Метод нанесения |
|-----|-------------|----------------------|-----------------|
| **T1** (Обычное) | +5% ~ +10% | Common+ | Свиток зачарования, NPC |
| **T2** (Усиленное) | +10% ~ +20% | Refined+ | Свиток + духовный камень |
| **T3** (Мощное) | +20% ~ +30% | Perfect+ | Мастер-зачарователь + камень+ |
| **T4** (Великое) | +30% ~ +40% | Perfect+ | Ритуал формации + камень кристалл |
| **T5** (Мифическое) | +40% ~ +50% | Transcendent | Артефакт зачарования + ядро дух. камня |

### 5.5.5 Правила зачарования

1. **Один слот зачарования** — предмет может иметь максимум 1 зачарование (T1-T3) или до 2 (T4-T5, но разных типов)
2. **Перезапись** — новое зачарование заменяет старое (старое теряется)
3. **Зависимость от грейда** — зачарование НЕ усиливает грейд и НЕ заменяет его. Это отдельный слой «Матрёшки»
4. **Множитель грейда** — сила зачарования тоже умножается на множитель эффективности грейда:
   ```
   effectiveEnchant = baseEnchantValue × gradeEfficiencyMultiplier
   ```
   Пример: Зачарование «Пылающий клинок» (+15% урона огнём) на Perfect грейде (×1.6) → +24%
5. **Совместимость** — стихийное зачарование может конфликтовать с innateElement практика (штраф -50% эффективности при несовпадении)

### 5.5.6 Нанесение и снятие

| Действие | Требования | Риск |
|----------|-----------|------|
| Нанесение зачарования | Свиток / NPC / формация + Ци практика | Шанс провала 10-30% (зависит от тира) |
| Снятие зачарования | Очищающий свиток или NPC-мастер | Безопасно, но зачарование теряется |
| Улучшение зачарования | Тот же тип, высший тир | Провал = потеря старого зачарования |

### 5.5.7 Примеры зачарованных предметов

**Пример 1: «Пылающий меч Небесного огня»**
```
База: Меч (sword), Steel T2, Refined
Грейд бонусы: +30% урона, ×1.3 эффективность
Зачарование: Пылающий клинок T2 (+15% урона огнём)
Итого: +30% физ. урон + (15% × 1.3) = +30% физ. + 19.5% огн.
```

**Пример 2: «Кольцо Духовного потока»**
```
База: Кольцо (ring), Jade T3, Perfect
Грейд бонусы: +60% qiConductivity, ×1.6 эффективность
Зачарование: Поток Ци T3 (+25% qiRestoration)
Итого: +60% qiConductivity + (25% × 1.6) = +60% cond. + 40% rest.
```

### 7.3 Формула урона оружия

**Гибридный метод:**

```
baseDamage = max(handDamage, weaponDamage × 0.5)
bonusDamage = weaponDamage × statScaling
totalDamage = baseDamage + bonusDamage
```

**Где:**
- handDamage = 3 + (STR-10) × 0.3
- statScaling = (STR×STR_BONUS + AGI×AGI_BONUS) / 100

### 7.4 Влияние на техники

Оружие с техникой melee_strike:
- Добавляет свой урон к урону техники
- Может иметь bonus к урону техник
- Может снижать стоимость Ци техник

## 8️⃣ БРОНЯ И ЗАЩИТА

### 8.1 Параметры брони

| Параметр | Описание | Диапазон |
|----------|----------|----------|
| armor | Базовая броня | 1-100 |
| damageReduction | Снижение урона % | 0-80% |
| coverage | Покрытие части тела | 30-100% |
| moveSpeedPenalty | Штраф скорости | -50% ~ 0% |
| dodgePenalty | Штраф уклонения | -40% ~ 0% |
| qiFlowPenalty | Штраф проводимости | -20% ~ +10% |

### 8.2 Штрафы веса

**Базовая вместимость** = 10 + (STR-10) × 2 кг

**Перегруз:**
- Каждый кг перегруза: -0.5% скорости, -0.3% уклонения, +1% стоимости выносливости

### 8.3 Проводимость Ци материалов

| Материал | Штраф к qi_efficiency |
|----------|----------------------|
| Ткань (cloth) | 0% |
| Кожа (leather) | -5% |
| Железо (iron) | -10% |
| Сталь (steel) | -15% |
| Холодное железо | -20% |
| Духовное железо | +5% |
| Нефрит (jade) | +10% |

### 8.4 Объём экипировки

Объём экипировки рассчитывается по формуле и определяет место в рюкзаке/хранилище.

| Категория | Формула volume | Пример |
|-----------|---------------|--------|
| Оружие | clamp(weight, 1, 4) | Меч 2.5кг → 2.5л |
| Броня | clamp(weight, 1, 4) | Шлем 2.5кг → 2.5л |
| Аксессуар | 0.2 (константа) | Кольцо → 0.2л |

**Примеры:**

| Предмет | weight (кг) | volume (л) |
|---------|-------------|------------|
| Кинжал | 0.5 | 1.0 (min) |
| Меч | 2.5 | 2.5 |
| Двуручник | 6.0 | 4.0 (max) |
| Тканевая роба | 0.5 | 1.0 (min) |
| Латный доспех | 8.0 | 4.0 (max) |
| Кольцо | 0.05 | 0.2 |
