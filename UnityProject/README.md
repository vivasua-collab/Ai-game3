# 🎮 Cultivation World Simulator

**Версия:** 0.1.0-alpha
**Unity:** 6000.3 (2D Core с URP)
**Платформа:** PC (Windows/Mac/Linux)

---

## 📖 Описание

Однопользовательская игра-симулятор культивации в жанре Xianxia. Игрок развивает своего персонажа от смертного до могущественного бессмертного.

### Ключевые особенности:
- 🌱 **Развитие смертного** → Пробуждение → Культивация
- 🔥 **10 уровней культивации** с уникальными способностями
- ⚔️ **Боевая система** с 10-слойным пайплайном урона
- 🧘 **Система Ци** с ядром культивации
- 🦴 **Система тела** Kenshi-style (двойная HP)
- 🤖 **NPC AI** с Behaviour Trees
- 🌍 **Живой мир** с течением времени

## 📁 Структура проекта

```
UnityProject/
├── Assets/
│   ├── Scripts/                    # C# скрипты
```

## 🔄 Синхронизация между ПК

### Правила:

1. ✅ **Всегда Pull перед началом работы**
2. ✅ **Закрывать Unity перед Push**
3. ✅ **Не редактировать одни файлы одновременно**
4. ❌ **НЕ пушить Library/, Temp/, Build/**

### Что синхронизируется:

| Папка | Git | Комментарий |
|-------|-----|-------------|
| Assets/ | ✅ | Вместе с .meta файлами |
| ProjectSettings/ | ✅ | Настройки проекта |
| Packages/ | ✅ | manifest.json |
| Library/ | ❌ | Кэш Unity (генерируется) |
| Temp/ | ❌ | Временные файлы |

## 🎯 Системы игры

### Путь развития

```
Смертный (L0) → Пробуждение → Практик (L1) → ... → Бессмертный (L9)
     ↑
     └── Дремлющее ядро формируется 16-30 лет ──┘
```

### Уровни культивации

| Уровень | Название | Плотность Ци |
|---------|----------|--------------|
| 0 | Смертный | — |
| 1 | Пробуждённое Ядро | ×1 |
| 2 | Течение Жизни | ×2 |
| 3 | Пламя Внутреннего Огня | ×4 |
| 4 | Объединение Тела и Духа | ×8 |
| 5 | Сердце Небес | ×16 |
| 6 | Разрыв Пелены | ×32 |
| 7 | Вечное Кольцо | ×64 |
| 8 | Глас Небес | ×128 |
| 9 | Бессмертное Ядро | ×256 |
| 10 | Вознесение | ×512 |

### Элементы

| Элемент | Противоположность | Сродство |
|---------|-------------------|----------|
| 🔥 Огонь | 💧 Вода | 💨 Воздух |
| 💧 Вода | 🔥 Огонь | ⚡ Молния |
| 🌍 Земля | 💨 Воздух | 🔥 Огонь |
| 💨 Воздух | 🌍 Земля | 🔥 Огонь, ⚡ Молния |
| ⚡ Молния | 🌍 Земля | 💧 Вода, 💨 Воздух |
| 🌑 Пустота | — | — |
