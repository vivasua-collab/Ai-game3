### Основные принципы

## 💥 Расчёт урона

### Базовый урон техники

**Формула:**
```
finalDamage = capacity × gradeMult
```

Где:
- `capacity` = затраченное Ци (ограничено ёмкостью техники)
- `gradeMult` = множитель Grade (×1.0 ~ ×1.6)

### Модификаторы урона

| Модификатор | Формула |
|-------------|---------|
| Мастерство | +0.5% за каждый % мастерства |
| Проводимость | +0.3% за каждую единицу проводимости |
| Уровень культивации | × множитель уровня |
| Стихийный бонус | Зависит от стихии |

## 🛡️ Буфер Ци

### Сырая Ци (без щитовой техники)

| Параметр | Значение |
|----------|----------|
| Поглощение | 90% урона |
| Стоимость | 3 Ци за 1 поглощённый урон |
| Пробитие | 10% урона ВСЕГДА |

### Щитовая техника

| Параметр | Значение |
|----------|----------|
| Поглощение | 100% урона |
| Стоимость | 1 Ци за 1 поглощённый урон |
| Пробитие | 0% |

### Примеры расчёта

**Атака 100 урона, 500 Ци:**

| Защита | Ци после | Поглощено | Урон в HP |
|--------|----------|-----------|-----------|
| Сырая Ци | 230 | 90 (270 Ци) | 10 |
| Щит | 400 | 100 (100 Ци) | 0 |
| Нет Ци | 0 | 0 | 100 |

## 🎯 Система попадания

### Уклонение

**Формула шанса:**
```
dodgeChance = базовый_шанс + (agilityAttacker - agilityDefender) × modifier
```

### Парирование

**Требования:**
- Имеется оружие или щит
- Цель в пределах досягаемости

**Эффект:** Снижение урона на % блока

## 📊 Подавление уровнем

### Проблема

Qi Buffer с поглощением 90% позволяет практике L1 наносить урон практике L8.

### Решение

Множитель подавления на основе разницы уровней:

| Разница уровней | Множитель урона |
|-----------------|-----------------|
| 0 | ×1.0 |
| 1 | ×0.5 |
| 2 | ×0.25 |
| 3 | ×0.1 |
| 4+ | ×0.01 |

## 🩸 Применение урона к телу

### Поток повреждения

1. **Функциональная HP** снижается первой
2. При 0 функциональной → урон идёт в структурную
3. При 0 структурной → часть отрублена

### Кровотечение

При значительном уроне начинается кровотечение:

| Степень | HP/тик |
|---------|--------|
| Лёгкое | 1 |
| Среднее | 3 |
| Тяжёлое | 5 |
| Критическое | 10 |

## 🎮 Механика боя

### Инициатива

Бой происходит в реальном времени с возможностью паузы.

### Действия в бою

| Действие | Стоимость времени |
|----------|-------------------|
| Атака | 1 тик |
| Использование техники | 2-5 тиков |
| Уклонение | 0.5 тика |
| Перемещение | 1 тик/клетка |

### Авто-переключение скорости

При начале боя автоматически включается `superSuperSlow` режим.
