## ⚔️ Система экипировки

### Архитектура "Матрёшка"

Три независимых слоя:

1. **Базовый класс** (неизменен)
   - Тип экипировки
   - Уровень предмета
   - Материал
   - Требования

2. **Материал** (неизменен)
   - Тир материала
   - Базовые свойства
   - Особые эффекты

3. **Grade** (изменяем)
   - Качество: Damaged → Common → Refined → Perfect → Transcendent
   - Множители параметров
   - Дополнительные бонусы

### Grade множители

| Grade | Прочность | Эффективность | Бонусов |
|-------|-----------|---------------|---------|
| Damaged | ×0.5 | ×0.5 | 0 |
| Common | ×1.0 | ×1.0 | 0-1 |
| Refined | ×1.5 | ×1.3 | 1-2 |
| Perfect | ×2.5 | ×1.7 | 2-4 |
| Transcendent | ×4.0 | ×2.5 | 4-6 |

### Распределение Grade

| Grade | Шанс |
|-------|------|
| Common | 60% |
| Refined | 28% |
| Perfect | 10% |
| Transcendent | 2% |

## 🛡️ Система прочности

### Состояния

| Состояние | Прочность | Эффективность |
|-----------|-----------|---------------|
| Pristine | 100% | 100% |
| Excellent | 80-99% | 95% |
| Good | 60-79% | 85% |
| Worn | 40-59% | 70% |
| Damaged | 20-39% | 50% |
| Broken | <20% | 20% |

### Потеря прочности

| Действие | Базовая потеря |
|----------|----------------|
| Атака | 0.1 |
| Блок | 0.2 |
| Поглощение урона | Зависит от урона |

### Ремонт

- Требует материалы
- Шанс понижения Grade при плохом ремонте
- Качество ремонта зависит от навыка

### Типы бонусов

| Категория | Бонусы |
|-----------|--------|
| Combat | Урон, крит, пробитие |
| Defense | Броня, уклонение, HP |
| Qi | Регенерация, проводимость |

| Поле | Описание |
|------|----------|

## 🎭 Сетовые бонусы

### Структура сета

| Поле | Описание |
|------|----------|
| id | ID сета |
| name | Название |
| pieces | ID предметов сета |
| setBonuses | Бонусы за количество |

### Пример

```
Fire Lord Set:
  Pieces: 5
  2 pieces: +20% сопротивление огню
  3 pieces: +5 интеллект, техника fire_aura
  5 pieces: +15 интеллект, +80% сопротивление огню
```
