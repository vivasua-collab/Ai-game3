## 🎭 Категории NPC

| Тип | Поведение | Сохранение |
|-----|-----------|------------|
| Temp | Упрощённое AI | Только в памяти |
| Plot | Полное AI | Сохранение в файл |
| Unique | Полное AI + история | Полное сохранение |

---

## 🔄 Машина состояний

### Переходы между состояниями

| Текущее | Условие | Следующее |
|---------|---------|-----------|
| idle | Игрок в радиусе агрессии | chase |
| idle | Точка патруля | patrol |
| patrol | Игрок в радиусе агрессии | chase |
| chase | Цель в радиусе атаки | attack |
| chase | Цель потеряна | patrol |
| attack | HP < 20% | flee |
| attack | Цель мертва | patrol |
| flee | В безопасности | patrol |
| flee | HP = 0 | dead |

## ⚡ Spinal AI (Рефлексы)

### Концепция

Быстрые автоматические реакции без сознательного решения.

### Типы рефлексов

| Рефлекс | Триггер | Действие |
|---------|---------|----------|
| Dodge | Снаряд приближается | Уклонение |
| Flinch | Получен урон | Отшатывание |
| Flee | HP < порог | Бегство |
| QiShield | Атака Ци | Авто-щит |
| Alert | Движение рядом | Настороженность |

### Параметры рефлексов

| Параметр | Описание |
|----------|----------|
| triggerDistance | Расстояние срабатывания |
| reactionTime | Время реакции |
| cooldown | Перезарядка |
| qiCost | Стоимость Ци |

## 🧠 Behaviour Tree

### Узлы дерева

| Тип | Описание |
|-----|----------|
| Selector | Выбирает первую успешную ветку |
| Sequence | Выполняет все узлы по порядку |
| Condition | Проверка условия |
| Action | Выполнение действия |

### Пример дерева

```
Selector
├── Sequence (Combat)
│   ├── Condition: InCombat?
│   ├── Condition: HasTarget?
│   └── Action: Attack
│
├── Sequence (Alert)
│   ├── Condition: IsAlert?
│   └── Action: SearchArea
│
├── Sequence (Patrol)
│   ├── Condition: HasPatrolRoute?
│   └── Action: Patrol
│
└── Action: Idle
```

## 📊 Параметры AI

### Базовые параметры

| Параметр | Описание | Диапазон |
|----------|----------|----------|
| aggression | Агрессивность | -100 до 100 |
| awareness | Осведомлённость | 0 до 100 |
| courage | Смелость | 0 до 100 |
| intelligence | Интеллект | 0 до 100 |

### Радиусы

| Радиус | Описание |
|--------|----------|
| aggroRadius | Радиус агрессии |
| attackRadius | Радиус атаки |
| patrolRadius | Радиус патруля |
| fleeRadius | Радиус бегства |

## 🎯 Целевая система

### Выбор цели

1. **Проверка aggroRadius**
2. **Оценка угрозы** (уровень, поведение)
3. **Приоритеты** (класс, раса, фракция)
4. **Выбор основной цели**

### Переключение цели

| Условие | Действие |
|---------|----------|
| Текущая цель мертва | Выбрать новую |
| Более опасная цель | Переключиться |
| HP < 20% | Бегство |

## 💬 Социальное взаимодействие

### Расчёт disposition

```
finalDisposition = 
  + personalDisposition
  + sectRelation × 0.5
  + factionRelation × 0.3
  + nationRelation × 0.2
  + attireModifier
  + reputationModifier
```

### Пороги disposition

| Значение | Отношение | Поведение |
|----------|-----------|-----------|
| 80-100 | Дружелюбный | Помощь, торговля |
| 50-79 | Благосклонный | Мирный диалог |
| 20-49 | Нейтральный | Осторожность |
| -20-19 | Подозрительный | Избегание |
| -50--21 | Враждебный | Агрессия |
| -100--51 | Ненависть | Атака без предупреждения |
