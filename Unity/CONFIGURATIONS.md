### 1.2 Способности по уровням

**Уровень 1 — Пробуждённое Ядро:**
- Первичное накопление Ци в ядре
- Тело становится чуть выносливее

**Уровень 2 — Течение Жизни:**
- Ускоренное заживление (порезы за час)
- Лёгкое усиление мышц
- Интуитивное чувство опасности

**Уровень 3 — Пламя Внутреннего Огня:**
- Регуляция температуры тела волей
- Очищение воды/воздуха от примесей Ци
- Доступ к простым боевым техникам Ци

**Уровень 4 — Объединение Тела и Духа:**
- Физические параметры превосходят человеческие пределы
- Кратковременное "парение"

**Уровень 5 — Сердце Небес:**
- Слушание Ци ландшафта
- Эмоциональная стабильность

**Уровень 6 — Разрыв Пелены:**
- Искажение локального поля Ци
- Тело почти неуязвимо для обычного оружия
- Способность жить без еды/воды

**Уровень 7 — Вечное Кольцо:**
- Пассивное поглощение Ци даже во сне
- Старение остановлено

**Уровень 8 — Глас Небес:**
- Предсказание природных катаклизмов
- Регенерация при потере конечностей

**Уровень 9 — Бессмертное Ядро:**
- Почти бессмертие
- Создание духовных артефактов с волей

### 1.3 Формулы культивации

**Плотность Ци:**
```
qiDensity = 2^(level - 1)
```

**Рост ёмкости ядра:**
```
newCapacity = currentCapacity × 1.1
```

**Ци для прорыва:**
```
// Малый уровень (под-уровень)
requiredQi = coreCapacity × 10

// Большой уровень (основной)
requiredQi = coreCapacity × 100
```

**Генерация микроядром:**
```
dailyGeneration = coreCapacity × 0.1  // 10% от ёмкости в сутки
```
