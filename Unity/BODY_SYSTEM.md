## 📋 Обзор

Система тела предназначена для конструирования тел всех сущностей мира:
- **Игроки** — при генерации персонажа
- **NPC** — неигровые персонажи
- **Монстры** — враждебные существа
- **Духи** — нематериальные сущности

### Цели системы

1. **Модульность** — сборка любого существа из компонентов
2. **Наследование** — свойства вида влияют на тело
3. **Секторные повреждения** — хитбоксы частей тела (Kenshi-style)
4. **Масштабируемость** — от мыши до титана

## 🏗️ Иерархия типов сущностей

### Уровень 1: SoulType (Природа сущности)

Определяет фундаментальные свойства существа.

| SoulType | Тело | Ци | Сознание | Примеры |
|----------|------|-----|----------|---------|
| Character | Органическое | Ядро | Полное | Человек, Эльф, Демон |
| Creature | Органическое | Ядро | Инстинкты | Волк, Тигр, Дракон |
| Spirit | Эфирное | Резервуар | Полное | Призрак, Элементаль |
| Construct | Конструкция | Резервуар | Простое | Голем |

### Уровень 2: Morphology (Морфология)

Определяет внешнюю форму и набор частей тела.

| Morphology | Описание | Части тела | Примеры |
|------------|----------|------------|---------|
| Humanoid | Двурукое двуногое | 11 + сердце | Человек, Эльф, Демон |
| Quadruped | Четвероногое | 8 + сердце | Волк, Тигр, Медведь |
| Bird | Крылатое | 6-7 + сердце | Орёл, Феникс |
| Serpentine | Змееподобное | 6 + сегменты | Змея, Ламия |
| Arthropod | Членистоногое | Экзоскелет | Паук, Скорпион |
| Amorphous | Бесформенное | 2 (core + essence) | Призрак, Элементаль |
| Hybrid_* | Гибридные формы | Зависит от типа | Кентавр, Русалка |

### Уровень 3: Species (Вид)

Конкретный вид с уникальными характеристиками.

**Примеры для SoulType=Character:**
- Human (Morphology=Humanoid)
- Elf (Morphology=Humanoid)
- Demon (Morphology=Humanoid)
- Giant (Morphology=Humanoid, Size=Huge)

**Примеры для SoulType=Creature:**
- Wolf (Morphology=Quadruped)
- Dragon (Morphology=Quadruped, Material=Scaled)
- Phoenix (Morphology=Bird, Material=Ethereal)

## 💖 Система двойной HP (Kenshi-style)

### Концепция

Каждая часть тела имеет два типа HP:
- **Функциональная HP (красная)** — работоспособность
- **Структурная HP (чёрная)** — целостность

### Визуализация

```
┌─────────────────────────────────────────────────────────────────┐
│                    HP КОНЕЧНОСТИ (Пример: Рука)                 │
├─────────────────────────────────────────────────────────────────┤
│   ████████████████████░░░░░░░░░░░░░░░░░░░░  Красная HP (40)    │
│   │←── Функциональная HP ──→│              (паралич при 0)     │
│                                                                 │
│   ████████████████████████████████████████░░░░░░░░░░░░░░░░░░░░ │
│   │←────────── Структурная HP (80) ──────────────→│            │
│                                                   (отрубание)   │
└─────────────────────────────────────────────────────────────────┘
```

### Соотношение HP

**Правило:** Структурная HP = Функциональная HP × 2

**Исключение:** Сердце имеет только функциональную HP

### Механика повреждения

**Порядок:**
1. Урон сначала снижает функциональную HP
2. При достижении 0 функциональной — урон идёт в структурную
3. При 0 структурной — часть отрублена

**Состояния части тела:**
- Healthy — оба типа HP полны
- Damaged — функциональная HP снижена
- Crippled — функциональная HP = 0, часть парализована
- Severed — структурная HP = 0, часть отрублена

### Иерархия зависимостей

```
torso
├── head
├── left_arm → left_hand
├── right_arm → right_hand
├── left_leg → left_foot
└── right_leg → right_foot
```

**Правило:** При потере родительской части все дочерние теряются.

## 💪 Живучесть (Vitality)

### Влияние на HP

Живучесть — характеристика, увеличивающая HP всех частей тела.

**Формула множителя:**
Базовый множитель 1.0 + 0.05 за каждую единицу живучести выше 10.

| Vitality | HP множитель | Регенерация | Сопротивление шоку |
|----------|--------------|-------------|---------------------|
| 10 | ×1.0 | 0% | 0% |
| 20 | ×1.5 | +10% | +5% |
| 30 | ×2.0 | +20% | +10% |
| 50 | ×3.0 | +40% | +20% |
| 75 | ×4.25 | +65% | +35% |
| 100 | ×5.5 | +90% | +50% |

### Развитие живучести

| Источник | Прирост | Примечание |
|----------|---------|------------|
| Получение урона + восстановление | +0.005 | Медленное лечение |
| Техники закалки тела | +0.02-0.05 | Специальные методы |
| Лекарственные ванны | +0.01-0.03 | Алхимия |

## 📈 Развитие физических характеристик

### Принцип линейного развития

**Из лора:** "нет стадии куколки, чтобы резко делать скачки"

**Решение:** Развитие физических характеристик (сила, ловкость):
1. **Постепенное** — прирост 0.001 за действие
2. **Накопительное** — виртуальная дельта накапливается
3. **Закрепляемое** — сон конвертирует виртуальную дельту в реальную
4. **С порогами** — чем выше стат, тем больше дельты нужно

### Виртуальная дельта развития

```
┌─────────────────────────────────────────────────────────────────┐
│                    СИСТЕМА ВИРТУАЛЬНОЙ ДЕЛЬТЫ                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   Физическая нагрузка (тренировка/бой)                          │
│       │                                                          │
│       ▼                                                          │
│   ┌───────────────────────────────────────────────┐             │
│   │ ВИРТУАЛЬНАЯ ДЕЛЬТА (временная)                │             │
│   │                                               │             │
│   │ virtualStrengthDelta: +0.047                  │             │
│   │ virtualAgilityDelta: +0.023                   │             │
│   │                                               │             │
│   │ НЕ влияет на боевые характеристики!           │             │
│   │ НЕ теряется без сна (кап на дельту)           │             │
│   └───────────────────────────────────────────────┘             │
│       │                                                          │
│       │ Минимум 4 часа сна                                       │
│       ▼                                                          │
│   ┌───────────────────────────────────────────────┐             │
│   │ ЗАКРЕПЛЕНИЕ (конвертация)                     │             │
│   │                                               │             │
│   │ Сон 4+ часов:                                │             │
│   │ - Максимум за сон: +0.20 (8 часов)           │             │
│   │ - Остаток сохраняется в виртуальной дельте   │             │
│   └───────────────────────────────────────────────┘             │
│       │                                                          │
│       ▼                                                          │
│   ┌───────────────────────────────────────────────┐             │
│   │ РЕАЛЬНАЯ ХАРАКТЕРИСТИКА                       │             │
│   │                                               │             │
│   │ strength: 12.047 → 12.067 (закреплено +0.02)  │             │
│   └───────────────────────────────────────────────┘             │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Источники прироста

| Действие | Прирост | Характеристика |
|----------|---------|----------------|
| Удар в бою | +0.001 | Сила |
| Уклонение/блок | +0.001 | Ловкость |
| Физическая тренировка | +0.002/мин | По выбору |
| Использование техники | +0.0005 | Интеллект |
| Получение урона + лечение | +0.005 | Живучесть |
| Медитация | +0.01/мин | Интеллект |

### Система порогов развития

**Принцип:** Аналогия с ядром культивации — чем больше характеристика, тем больше "виртуальной дельты" нужно для повышения.

**Формула порога:** Базовый порог равен 1.0, затем увеличивается с ростом характеристики.

**Примеры:**
- Стат 10 → 11: нужно 1.0 виртуальной дельты
- Стат 20 → 21: нужно 2.0 виртуальной дельты
- Стат 50 → 51: нужно 5.0 виртуальной дельты
- Стат 100 → 101: нужно 10.0 виртуальной дельты

### Закрепление при сне

**Правила:**
- Минимум 4 часа сна для закрепления
- Максимум закрепления за сон: +0.20 (при 8 часах)
- 4 часа = +0.067, 6 часов = +0.133, 8 часов = +0.20

### Достижимость за периоды

| Период | Достижимый стат | Время на +1 |
|--------|-----------------|-------------|
| 1000 дней | ~55 | 21-28 дней |
| 3000 дней | ~80 | 42-56 дней |
| 10000 дней | ~125 | 70+ дней |

## ⚡ Буфер Ци как защита

### Концепция

Ци действует как "энергетический щит", поглощая урон от техник Ци.

### Сырая Ци (без щитовой техники)

- **Поглощение:** 90% урона от техник Ци
- **Стоимость:** 3 единицы Ци за 1 поглощённый урон
- **Пробитие:** 10% урона ВСЕГДА проходит к HP

### Щитовая техника

- **Поглощение:** 100% урона
- **Стоимость:** 1 единица Ци за 1 поглощённый урон
- **Поддержание:** Расход Ци в секунду

### Примеры

| Сценарий | Урон | Ци до | Ци после | Поглощено | Пробитие в HP |
|----------|------|-------|----------|-----------|---------------|
| Сырая Ци, достаточно | 100 | 500 | 230 | 90 (270 Ци) | **10** |
| Сырая Ци, мало | 100 | 150 | 0 | 50 (150 Ци) | **50** |
| Щит, достаточно | 100 | 500 | 400 | 100 (100 Ци) | **0** |
| Щит, мало | 100 | 50 | 0 | 50 (50 Ци) | **50** |
| Смертный (нет Ци) | 100 | 0 | 0 | 0 | **100** |

## 🔗 Приживление конечностей

### Концепция

Возможность приживить чужую конечность взамен утраченной.

### Условия

1. Совместимость по размеру (в пределах одного класса)
2. Совместимость по материалу (organic → organic)
3. Хирургическая операция (требуется врач)
4. Время приживления (зависит от качества)

### Риски

- **Отторжение** — несовместимость по виду
- **Инфекция** — плохая стерильность
- **Нервное несоответствие** — снижение ловкости

## 🩸 Кровотечение и регенерация

### Кровотечение

При получении значительного урона начинается кровотечение:
- **Лёгкое** — 1 HP/тик
- **Среднее** — 3 HP/тик
- **Тяжёлое** — 5 HP/тик
- **Критическое** — 10 HP/тик

### Регенерация

**Базовая скорость:** 0.1 HP/тик для органического материала

**Порядок восстановления:**
1. Сначала структурная HP
2. Затем функциональная HP

**Факторы ускорения:**
- Высокая живучесть (+% регенерации)
- Лекарственные средства
- Техники исцеления
- Отдых
