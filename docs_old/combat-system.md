## 1️⃣ Классификация техник

### Типы техник

| Тип | Префикс | Описание |
|-----|---------|----------|
| combat | TC | Атакующие техники |
| **defense** | DF | Защитные техники |
| cultivation | CU | Культивация |
| support | SP | Поддержка (баффы) |
| movement | MV | Перемещение |
| sensory | SN | Восприятие |
| healing | HL | Исцеление |
| **curse** | CR | Проклятия |
| **poison** | PN | Отравления |

### Подтипы

**Defense:** shield, barrier, block, dodge, absorb, reflect
**Curse:** combat (секунды-минуты), ritual (часы-месяцы)
**Poison:** body (физическое тело), qi (меридианы и ядро)

## 2️⃣ Механика наполнения Ци

**У техник нет отката (cooldown).** Вместо этого — система наполнения:

```
Активация → Накопление Ци → Порог достигнут → Мгновенное применение
                │
                ▼
     Скорость = проводимость ед/сек
```

### Формула времени

```
время_наполнения = qiCost / проводимость (секунды)
```

| Стоимость Ци | Проводимость 1.0 | Проводимость 5.0 |
|--------------|------------------|------------------|
| 5 Ци | 5 сек | 1 сек |
| 15 Ци | 15 сек | 3 сек |
| 50 Ци | 50 сек | 10 сек |

### Модификаторы скорости

```typescript
effectiveSpeed = conductivity * (1 + cultivationLevel * 0.05) * (1 + mastery * 0.01)
```

## 4️⃣ Защитные техники (Defense)

### Типы защиты

| Подтип | Механика | Пример |
|--------|----------|--------|
| shield | Поглощение урона за счёт Ци | "Водяной барьер" |
| block | Снижение урона на % | "Железная стена" |
| dodge | Шанс уклонения | "Тень призрака" |
| reflect | Отражение урона | "Зеркальный щит" |

### Параметры

```typescript
interface DefenseEffects {
  damageReduction?: number;   // Снижение урона (%)
  shieldHP?: number;          // Здоровье щита
  dodgeChance?: number;       // Шанс уклонения (%)
  counterBonus?: number;      // Бонус к контратаке (%)
  duration: number;           // Длительность
}
```

## 5️⃣ Проклятия (Curse)

### Боевые проклятия

- Длительность: секунды-минуты
- Применение: мгновенно в бою
- Слот: отдельный curseSlot

**Типы эффектов:** weakness, slowness, blindness, silence, confusion, fear, exhaustion, qi_drain, soul_burn

### Ритуальные проклятия

- Длительность: часы-месяцы
- Применение: через ритуал/формацию
- Снятие: только ритуалом очищения

**Типы эффектов:** cultivation_block, meridian_damage, core_corruption, soul_seal, karma_curse

---

## 6️⃣ Отравления (Poison)

### Яды тела (Body Poison)

- Воздействует на физическое тело
- Доставка: ingestion, contact, injection, inhalation
- Стадии: onset → progression → critical

### Яды Ци (Qi Poison)

- Воздействует на меридианы и ядро
- Доставка: technique, contaminated_qi, cursed_item
- Только для культиваторов

## 7️⃣ Сопротивление

### Проклятиям

```typescript
resistance = cultivationLevel * 5 + willpower * 2 + conductivity * 10
// Диапазон: 0-95%
```

### Ядам

```typescript
// Body Poison
resistance = constitution * 2 + cultivationLevel * 8

// Qi Poison
resistance = conductivity * 20 + cultivationLevel * 10
```

## 8️⃣ Слоты техник

```typescript
interface TechniqueSlots {
  cultivationSlot: string | null;     // 1 слот
  combatSlots: (string | null)[];     // 3 + (level-1) слотов
  curseSlot: string | null;           // 1 слот для проклятия
  formationSlot: string | null;       // 1 слот для формации
}
```

**Почему проклятия в отдельном слоте:**
1. Баланс — игрок не может загрузить все слоты проклятиями
2. Тактика — выбор между проклятием и защитой
3. Лор — проклятия требуют особого канала Ци
