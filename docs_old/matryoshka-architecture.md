## 📋 Суть архитектуры

**Матрёшка** — принцип многослойного построения объектов, где каждый слой добавляет свойства к базовому объекту.

```
┌─────────────────────────────────────────────────────────────────┐
│                      ИТОГОВЫЙ ОБЪЕКТ                             │
│                                                                  │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │              СЛОЙ 3: СПЕЦИАЛИЗАЦИЯ                       │   │
│   │   (тип оружия, слот брони, подтип техники)              │   │
│   └─────────────────────────────────────────────────────────┘   │
│                          ×                                       │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │              СЛОЙ 2: ГРЕЙД (Grade)                       │   │
│   │   Common → Refined → Perfect → Transcendent             │   │
│   └─────────────────────────────────────────────────────────┘   │
│                          ×                                       │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │              СЛОЙ 1: БАЗА (Base)                         │   │
│   │   Уровень → базовые параметры (qiCost, damage)          │   │
│   └─────────────────────────────────────────────────────────┘   │
│                                                                  │
│   Результат = Base × Grade × Specialization                     │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

---

## 🎯 Ключевые принципы

### 1. Независимость слоёв

Каждый слой **НЕ зависит** от других:
- **База** — определяет материальные свойства
- **Грейд** — определяет качество, множители
- **Специализация** — определяет конкретный тип

### 2. Детерминированность

Все генераторы используют **seededRandom**:
```typescript
const rng = seededRandom(seed);
// При одинаковом seed — одинаковый результат
```

### 3. Унификация Grade

**Grade НЕ зависит от уровня!**

```
Даже на L1 есть шанс получить transcendent:
- common:      60%
- refined:     28%
- perfect:     10%
- transcendent: 2%
```

## 📊 Применение по типам объектов

### Экипировка

| Слой | Что определяет | Пример |
|------|----------------|--------|
| База | Материал → базовые статы | Железо → 10 урона |
| Грейд | Множители | Refined → ×1.3 урона |
| Специализация | Тип предмета | Sword → critChance +5% |

**Итог:** `Урон = (База железа) × (Грейд ×1.3) + (Меч +crit)`

### Техники (V2)

| Слой | Что определяет | Пример |
|------|----------------|--------|
| База | Уровень → qiCost = baseDamage | L5 → qiCost=50, damage=50 |
| Грейд | Множитель урона | Perfect → ×1.4 |
| Специализация | Подтип техники | melee_strike → ×1.5 |

**Итог:** `Урон = qiCost × (Грейд ×1.4) × (Подтип ×1.5)`

**Пример L5 Perfect melee_strike:**
```
baseDamage = qiCost = 50 (Слой 1)
gradeMult = 1.4 (Слой 2)
specMult = 1.5 (Слой 3)
finalDamage = 50 × 1.4 × 1.5 = 105
```

> **КЛЮЧЕВОЙ ПРИНЦИП V2:** `baseDamage = qiCost` (соотношение 1:1)
> Уровень техники НЕ добавляет урон напрямую!
> Уровень влияет на ёмкость (сколько Ци можно безопасно использовать)

### Расходники

| Слой | Что определяет | Пример |
|------|----------------|--------|
| База | Уровень → базовое значение | L3 → 35 HP |
| Грейд | Множитель эффекта | Perfect → ×1.7 |
| Специализация | Тип расходника | Pill → мгновенное действие |

**Итог:** `Эффект = (База 35) × (Грейд ×1.7) = 59.5 HP`

## 🔧 Формула расчёта

### Универсальная формула

```typescript
function calculateFinalValue(base: number, grade: Grade, spec?: Spec): number {
  let value = base;
  
  // Слой 1: База уже применена
  
  // Слой 2: Грейд
  value *= GRADE_MULTIPLIERS[grade];
  
  // Слой 3: Специализация
  if (spec) {
    value *= spec.multiplier;
    value += spec.bonus;
  }
  
  return Math.floor(value);
}
```

### Пример для оружия

```typescript
// База: Железо → 10 урона
const baseDamage = 10;

// Грейд: Refined → ×1.3
const gradeMult = 1.3;

// Специализация: Sword → crit +5%, damage +0%
const specBonus = 0;

// Итог
const damage = Math.floor(10 * 1.3 + 0); // = 13
```

### Пример для техники V2

```typescript
// База: L5 техника → qiCost = 50, baseDamage = qiCost
const baseDamage = calculateBaseQiCost(5); // 50

// Грейд: Perfect → ×1.4
const gradeMult = GRADE_DAMAGE_MULTIPLIERS.perfect; // 1.4

// Специализация: melee_strike → ×1.5
const specMult = COMBAT_SUBTYPE_DAMAGE_MULTIPLIERS.melee_strike; // 1.5

// Итог
const damage = Math.floor(50 * 1.4 * 1.5); // = 105
```

> **Формула qiCost:** `qiCost = 10 × 1.5^(level-1)`
>
> | L1 | L3 | L5 | L7 | L9 |
> |----|----|----|----|----|
> | 10 | 22 | 50 | 112 | 250 |
