## ❌ Проблема текущей формулы

Из чекпоинта P0-1:
```
damage = baseDamage × (1 + qiDensity/100)
```

### Результат:
| Уровень | qiDensity | Множитель урона |
|---------|-----------|-----------------|
| 1       | 1         | ×1.01           |
| 3       | 4         | ×1.04           |
| 5       | 16        | ×1.16           |
| 7       | 64        | ×1.64           |
| 9       | 256       | ×3.56           |

**Проблема:** Практик L9 всего в 3.56 раза сильнее L1!  
**Ожидание:** Геометрический рост ×2 за уровень → L9 в 256 раз сильнее L1.

---

## ✅ Решение: Формула из Lore

Из `start_lore.md` секция 200:
```
Эффективность = Потрачено Ци × Качество Ци
Качество Ци = qiDensity = 2^(уровень-1)
```

### Новая формула:

```typescript
/**
 * Расчёт урона техники
 * 
 * @param qiSpent - Потрачено единиц Ци
 * @param cultivationLevel - Уровень культивации (1-9)
 * @returns Эффективность в базовых единицах
 */
function calculateTechniqueEffect(qiSpent: number, cultivationLevel: number): number {
  const qiDensity = Math.pow(2, cultivationLevel - 1);
  return qiSpent * qiDensity;
}
```

## ⚖️ Баланс: Стоимость техник

### Вариант A: Фиксированная стоимость

Техника всегда стоит X Ци, независимо от уровня.

```
Техника "Огненный шар" = 50 Ци

L1 практик:
  - Затраты: 50 Ци (5% от запаса 1000)
  - Урон: 50 × 1 = 50

L9 практик:
  - Затраты: 50 Ци (0.01% от запаса 500,000)
  - Урон: 50 × 256 = 12,800
```

**Проблема:** L9 тратит ничтожную часть запаса.

### Вариант B: Масштабируемая стоимость ✅ (рекомендуется)

Игрок может вложить больше Ци для усиления эффекта.

```typescript
interface ScalableTechnique {
  baseQiCost: number;       // Минимальная стоимость
  maxQiMultiplier: number;  // Максимальный множитель (напр. 10x)
}

// Пример: "Огненный шар"
// baseQiCost: 50
// maxQiMultiplier: 10

// L1 практик, вложил 50 Ци (множитель 1x):
// Урон = 50 × 1 × 1 = 50

// L9 практик, вложил 500 Ци (множитель 10x):
// Урон = 50 × 10 × 256 = 128,000
```

## 📐 Полная формула урона

```typescript
/**
 * Расчёт итогового урона техники
 */
function calculateTechniqueDamage(
  technique: Technique,
  cultivator: Cultivator,
  qiInvested: number,        // Вложенное Ци (базовое или масштабированное)
  modifiers: DamageModifiers
): number {
  
  // 1. Качество Ци (геометрический рост)
  const qiDensity = Math.pow(2, cultivator.level - 1);
  
  // 2. Базовый эффект = Ци × Качество
  let effect = qiInvested * qiDensity;
  
  // 3. Масштабирование от характеристик
  // (сила/ловкость/интеллект в зависимости от типа техники)
  const statMultiplier = calculateStatScaling(cultivator, technique.combatType);
  effect *= statMultiplier;
  
  // 4. Мастерство техники (0-100%)
  const masteryBonus = 1 + (technique.mastery / 100) * 0.3; // до +30%
  effect *= masteryBonus;
  
  // 5. Элементальные модификаторы
  if (modifiers.elementalBonus) {
    effect *= modifiers.elementalBonus;
  }
  
  return Math.floor(effect);
}
```

## 🛡️ Формула поглощения урона

```typescript
/**
 * Расчёт входящего урона с учётом защиты
 */
function calculateIncomingDamage(
  rawDamage: number,
  target: Cultivator,
  technique: Technique
): number {
  
  // 1. meridianBuffer — быстрый буфер Ци в меридианах
  //    Формула из Lore: запас = проводимость × 5 сек
  const meridianBuffer = target.conductivity * 5;
  
  // 2. Поглощение буфером (если Ци есть в меридианах)
  let remainingDamage = rawDamage;
  const bufferedDamage = Math.min(meridianBuffer, rawDamage * 0.3); // до 30%
  remainingDamage -= bufferedDamage;
  
  // 3. Физическая броня
  const armorReduction = target.armor * 0.5; // 50% эффективности брони
  remainingDamage = Math.max(1, remainingDamage - armorReduction);
  
  // 4. Разница в уровнях (высшие уровни игнорируют защиту низших)
  const levelDiff = attackerLevel - targetLevel;
  if (levelDiff > 0) {
    const penetrationBonus = levelDiff * 0.1; // +10% пробития за уровень
    remainingDamage *= (1 + penetrationBonus);
  }
  
  return Math.floor(remainingDamage);
}
```

## 📊 Пример боя: L3 vs L5

```
Атакующий: L5 практик
  - qiDensity: 16
  - Техника: "Огненный шар" (база 50 Ци)
  - Вложено Ци: 100 (множитель 2x)

Защитник: L3 практик  
  - qiDensity: 4
  - Проводимость: 10 ед/сек
  - meridianBuffer: 50 Ци

Расчёт:
  1. Урон = 100 × 16 = 1,600 (эффективность)
  2. × стат множитель (напр. 1.2) = 1,920
  3. × мастерство (напр. 1.1) = 2,112

  4. Поглощение меридиан: min(50, 2112 × 0.3) = 50
     Урон = 2,112 - 50 = 2,062
  
  5. Броня (напр. 20): 2062 - 10 = 2,052

  6. Пробитие уровня: L5 - L3 = 2
     +20% урона = 2,052 × 1.2 = 2,462

Итог: 2,462 урона (критический удар для L3)
```
