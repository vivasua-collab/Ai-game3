## ✅ Этап 9: Система хитбоксов

**Статус:** ✅ ЗАВЕРШЁН

### Реализованные функции

- **hitboxRadius** для целей и игрока
- **Эффективная дальность** = дальность + радиус хитбокса
- **Визуальные индикаторы** хитбоксов (зелёные круги)
- **Расстояние до края тела** (не до центра)
- **Регистрация попаданий** с учётом размера тела

### Константы

```typescript
PLAYER_HITBOX_RADIUS = 24;       // ~0.75 м
TARGET_HITBOX_RADIUS = 22;       // ~0.69 м
```

### Формула попадания

```typescript
// Атака достигает цели если:
distance <= range + targetHitboxRadius

// Эффективное расстояние:
effectiveDistance = distanceToCenter - hitboxRadius
```

### Созданные файлы

```

├── time-scaling.ts        ✅ Функции масштабирования
├── action-speeds.ts       ✅ Профили активностей
└── activity-manager.ts    ✅ Менеджер переключения
```

### Скорости

| Speed | Мин/тик | Factor | Восприятие |
|-------|---------|--------|------------|
| superSuperSlow | 0.25 | 20.0 | Бой (очень медленно) |
| superSlow | 0.5 | 10.0 | Точный режим |
| slow | 1 | 5.0 | Медленно |
| normal | 5 | 1.0 | Обычная скорость |
| fast | 15 | 0.333 | Путешествие |
| ultra | 60 | 0.083 | Медитация |

### Исправленная проблема

⚠️ **Инверсия скорости исправлена** (2026-03-24):
- Использована функция `scaleMovementSpeedInverse()`
- Теперь: slow time = slow movement, fast time = fast movement
