## 2️⃣ ⭐ Система Подавления Уровнем (Level Suppression)

### 2.1 Концепция

**Проблема:** Qi Buffer с поглощением 90% позволяет практику L1 наносить урон практику L8.

**Решение:** Ввести множитель подавления на основе разницы уровней культивации.

### 2.2 Таблица подавления

```typescript
// src/lib/constants/level-suppression.ts

export const LEVEL_SUPPRESSION_TABLE = {
  0: { normal: 1.0, technique: 1.0, ultimate: 1.0 },    // Тот же уровень
  1: { normal: 0.5, technique: 0.75, ultimate: 1.0 },   // +1 уровень
  2: { normal: 0.1, technique: 0.25, ultimate: 0.5 },   // +2 уровня
  3: { normal: 0.0, technique: 0.05, ultimate: 0.25 },  // +3 уровня
  4: { normal: 0.0, technique: 0.0, ultimate: 0.1 },    // +4 уровня
  5: { normal: 0.0, technique: 0.0, ultimate: 0.0 },    // +5+ уровней = иммунитет
};

// Типы атак:
// - normal: Обычная атака без техники
// - technique: Атака техникой (technique.level влияет)
// - ultimate: Ultimate-техника (особый флаг isUltimate)
```

### 2.3 Формула подавления

```typescript
function calculateLevelSuppression(
  attackerLevel: number,
  defenderLevel: number,
  attackType: 'normal' | 'technique' | 'ultimate',
  techniqueLevel?: number
): number {
  // Для техник: можно "пробить" защиту выше уровнем
  let effectiveAttackerLevel = attackerLevel;
  if (attackType === 'technique' && techniqueLevel) {
    effectiveAttackerLevel = Math.max(attackerLevel, techniqueLevel);
  }
  
  const levelDiff = Math.max(0, defenderLevel - effectiveAttackerLevel);
  const clampedDiff = Math.min(5, levelDiff);
  
  return LEVEL_SUPPRESSION_TABLE[clampedDiff][attackType];
}
```

### 2.4 Примеры подавления

| Атакующий | Защитник | Тип атаки | Множитель | Урон 39,321 → |
|-----------|----------|-----------|-----------|---------------|
| L9 | L9 | normal | 1.0 | 39,321 |
| L9 | L9 | technique | 1.0 | 39,321 |
| L7 | L9 | normal | 0.0 | 0 |
| L7 | L9 | technique (L7) | 0.05 | 1,966 |
| L7 | L9 | technique (L8) | 0.25 | 9,830 |
| L5 | L9 | ultimate | 0.1 | 3,932 |
| L4 | L9 | ultimate | 0.0 | 0 |

### 2.5 Ultimate-техники

Ultimate-техники имеют специальный флаг `isUltimate: true`:
- Могут пробить защиту на 4+ уровней выше (10% урона)
- Полный иммунитет только при разнице 5+ уровней
- Редкие и мощные техники

### 2.6 AOE и смешанные группы

При AOE атаке подавление применяется **ИНДИВИДУАЛЬНО** к каждой цели:

```typescript
function processAOEAttack(
  attacker: Character,
  targets: Character[],
  technique: Technique
): Map<string, DamageResult> {
  const results = new Map();
  
  for (const target of targets) {
    // 1. Расчёт базового урона
    let damage = calculateBaseDamage(technique, attacker);
    
    // 2. Коэффициент расстояния (для AOE)
    const distanceFactor = getDistanceFactor(attacker, target);
    damage *= distanceFactor;
    
    // 3. ⭐ ПОДАВЛЕНИЕ УРОВНЕМ — ИНДИВИДУАЛЬНО!
    const suppression = calculateLevelSuppression(
      attacker.cultivationLevel,
      target.cultivationLevel,
      technique.isUltimate ? 'ultimate' : 'technique',
      technique.level
    );
    damage *= suppression;
    
    // 4. Остальные слои защиты...
    results.set(target.id, processRemainingLayers(damage, target));
  }
  
  return results;
}
```

## 3️⃣ Параметры Брони по Слоям

### 3.1 Базовые параметры (Слой 1: Базовый класс)

```typescript
interface ArmorBaseStats {
  // Базовая защита (зависит от уровня и типа)
  armor: number;           // L1: 5-15, L5: 25-50, L9: 60-120
  
  // Типы защиты
  resistances: {
    slashing: number;      // Рубящий урон
    piercing: number;      // Колющий урон
    blunt: number;         // Дробящий урон
    elemental: number;     // Стихийный урон
  };
  
  // Покрытие
  coverage: number;        // 30-100%
  
  // Требования
  requirements: {
    strength?: number;
    cultivationLevel?: number;
  };
}
```

### 3.4 Формула итоговой защиты

```typescript
function calculateEffectiveArmor(armor: Armor): number {
  const baseArmor = armor.baseStats.armor;
  const materialMult = armor.material.properties.defenseBonus / 100 + 1;
  const gradeMult = GRADE_CONFIGS[armor.grade].effectivenessMultiplier;
  
  // Состояние прочности
  const conditionMult = getConditionMultiplier(armor.durability);
  
  return Math.floor(baseArmor * materialMult * gradeMult * conditionMult);
}

function calculateDamageReduction(armor: Armor): number {
  const baseDR = armor.baseStats.damageReduction || 0;
  const materialDR = armor.material.properties.resistanceBonus?.physical || 0;
  const gradeDR = GRADE_CONFIGS[armor.grade].damageReductionBonus || 0;
  
  // Мягкий кап: максимум 80% снижения урона
  return Math.min(0.80, (baseDR + materialDR + gradeDR) / 100);
}
```

```typescript
interface WeightPenalties {
  speedPenalty: number;      // -% скорости
  dodgePenalty: number;      // -% уклонения
  staminaCostPenalty: number; // +% стоимости выносливости
}

function calculateWeightPenalties(totalWeight: number, characterSTR: number): WeightPenalties {
  const baseCapacity = 10 + (characterSTR - 10) * 2;  // кг
  const overload = Math.max(0, totalWeight - baseCapacity);
  
  return {
    speedPenalty: Math.min(50, overload * 0.5),         // до -50%
    dodgePenalty: Math.min(40, overload * 0.3),         // до -40%
    staminaCostPenalty: Math.min(100, overload * 1.0),  // до +100%
  };
}
```

## 9️⃣ Износ Брони при Блокировании

### 9.1 Формула потери прочности

```typescript
/**
 * Расчёт потери прочности брони при блокировании урона
 * 
 * Принцип: Броня "принимает на себя" часть урона и изнашивается
 */
function calculateDurabilityLoss(
  damageAbsorbedByArmor: number,  // Урон, который броня заблокировала
  armorHardness: number,          // Твёрдость материала (1-10)
  armorCondition: number          // Процент прочности (0-100)
): number {
  // Базовая потеря: 0.1% макс. прочности за каждую 1000 урона
  const baseLoss = damageAbsorbedByArmor / 10000;  // 10,000 урона = 1% потери
  
  // Твёрдость снижает износ (твёрдость 10 = 50% износа)
  const hardnessFactor = 1 - (armorHardness - 1) / 18;  // 1.0 → 0.94, 10.0 → 0.5
  
  // Повреждённая броня изнашивается быстрее
  const conditionFactor = 1 + (1 - armorCondition / 100) * 0.5;  // до ×1.5 при 0%
  
  // Итоговая потеря
  const totalLoss = baseLoss * hardnessFactor * conditionFactor;
  
  return totalLoss;  // В процентах от maxDurability
}
```

### 9.2 Пример износа

```
Броня: Нагрудник "Кираса Драконьей Чешуи"
• Твёрдость: 9 (кость дракона)
• Прочность: 510/510 (100%)
• Заблокированный урон: 18,000

Расчёт:
  baseLoss = 18,000 / 10,000 = 1.8%
  hardnessFactor = 1 - (9-1)/18 = 0.556
  conditionFactor = 1.0 (броня целая)
  
  totalLoss = 1.8% × 0.556 × 1.0 = 1.0%

Потеря: 1.0% × 510 = 5.1 прочности
Итог: 505/510 (Pristine → Pristine)

═══════════════════════════════════════════════════════════════════

При 10 ударах L9 техникой:
  Суммарный урон: 180,000
  Потеря: 10% × 0.556 = 5.56%
  
  Итог: 482/510 (Pristine → Excellent)
```

## 1️⃣1️⃣ ⭐ ПОДАВЛЕНИЕ УРОВНЕМ (Слой 2) — НОВОЕ v4.0

### 11.1 Принцип

> **Разница в уровнях культивации должна быть критической.**
> 
> Практик L1 не может нанести урон практику L8.

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                    ПОДАВЛЕНИЕ УРОВНЕМ — МЕХАНИКА v4.0                               │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                      │
│  levelDiff = defenderLevel - attackerLevel                                          │
│  breakthrough = techniqueLevel (техника L8 пробивает до 8 уровней разницы)          │
│  effectiveDiff = max(0, levelDiff - breakthrough)                                   │
│                                                                                      │
│  attackType:                                                                        │
│  • normal — обычная атака (без техники)                                             │
│  • technique — техника Ци                                                           │
│  • ultimate — секретная техника (флаг isUltimate: true)                             │
│                                                                                      │
└─────────────────────────────────────────────────────────────────────────────────────┘
```

### 11.2 Таблица подавления

| Diff | normal | technique | ultimate | Описание |
|------|--------|-----------|----------|----------|
| **0** | ×1.0 | ×1.0 | ×1.0 | Равные уровни |
| **+1** | ×0.5 | ×0.75 | ×1.0 | Защитник на 1 выше |
| **+2** | ×0.1 | ×0.25 | ×0.5 | Защитник на 2 выше |
| **+3** | ×0 | ×0.05 | ×0.25 | Практически иммунитет |
| **+4** | ×0 | ×0 | ×0.1 | Почти полный иммунитет |
| **+5+** | ×0 | ×0 | ×0 | **ПОЛНЫЙ ИММУНИТЕТ** |

### 11.3 Пробитие техникой

**Техника L8 пробивает до 8 уровней разницы!**

```
Пример 1: Атакующий L5, Защитник L8, Техника L5
  levelDiff = 8 - 5 = 3
  breakthrough = 5 (техника L5)
  effectiveDiff = max(0, 3 - 5) = 0
  → ×1.0 урона (нет подавления!)

Пример 2: Атакующий L5, Защитник L8, Техника L2
  levelDiff = 3
  breakthrough = 2
  effectiveDiff = max(0, 3 - 2) = 1
  → ×0.75 урона (technique)

Пример 3: Атакующий L1, Защитник L8, Техника L1
  levelDiff = 7
  breakthrough = 1
  effectiveDiff = max(0, 7 - 1) = 6
  → ×0 (ИММУНИТЕТ!)
```

### 11.4 Ultimate-техники

**Флаг `isUltimate: true` в технике:**

```typescript
interface TechniquePreset {
  // ...
  isUltimate?: boolean;  // true = секретная техника
}
```

| Атака | L1 → L5 | L5 → L8 | L7 → L9 |
|-------|---------|---------|---------|
| Обычная | ×0 | ×0 | ×0.5 |
| Техника L1 | ×0 | ×0 | ×0.25 |
| Техника L5 | ×0.1 | ×0.25 | ×0.5 |
| Техника L9 | ×1.0 | ×1.0 | ×1.0 |
| Ultimate L9 | ×1.0 | ×1.0 | ×1.0 |

### 11.5 Пример расчёта с подавлением

```
Атакующий: L5, Техника "Огненный шар" L5 (не ultimate)
Защитник: L8 в броне

1. Базовый урон техники L5: 1,024
2. Grade Perfect (×1.4): 1,433
3. ⭐ ПОДАВЛЕНИЕ:
   levelDiff = 8 - 5 = 3
   breakthrough = 5
   effectiveDiff = 0
   multiplier = ×1.0 (нет подавления)

6. Броня (45% DR): 143 × 0.55 = 79 урона

```
