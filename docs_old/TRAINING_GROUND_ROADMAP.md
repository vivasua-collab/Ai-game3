## ✅ Реализовано (v0.6.x - v0.7.x)

### Управление и интерфейс

| Механика | Статус | Описание |
|----------|--------|----------|
| WASD + мышь | ✅ | Движение и прицеливание |
| 6 техник слотов | ✅ | Клавиши 1-6 |
| Qi аура | ✅ | Визуальная индикация уровня Ци |
| Синхронизация React | ✅ | Через глобальные переменные |

### Боевая система

| Механика | Статус | Описание |
|----------|--------|----------|
| 6 мишеней | ✅ | Соломенные чучела с HP |
| Базовый удар | ✅ | Слот 1, melee, 0 Ци |
| Урон по мишеням | ✅ | Базовый конус атаки |
| Числа урона | ✅ | Всплывающие числа с цветами |
| Зоны урона | ✅ | fullDamage / halfDamage / falloff |
| Зарядка техник | ✅ | Через Event Bus + Qi списание |

### NPC система

| Механика | Статус | Описание |
|----------|--------|----------|
| NPC Viewer | ✅ | 3-колоночный макет (Статус/Техники/Экипировка) |
| Фильтрация по локации | ✅ | Показ NPC только текущей локации |
| Отображение локации | ✅ | Название текущей локации в заголовке |
| Тип NPC (реальность) | ✅ | Сюжетный/Сессионный/Временный/Локальный |
| Пресеты NPC | ✅ | Сюжетные персонажи |

### Event Bus

| Механика | Статус | Описание |
|----------|--------|----------|
| technique:use | ✅ | Использование техник |
| player:move | ✅ | Движение игрока |
| Qi списание | ✅ | Через TruthSystem |

## 🎮 Режимы тестирования полигона

### Режим 1: Свободный (текущий)

```
┌────────────────────────────────────────┐
│  Игрок + 6 мишеней                     │
│  - Бесконечные HP мишеней             │
│  - Тестирование урона                 │
│  - Тестирование техник                │
└────────────────────────────────────────┘
```

### Режим 2: Волны (план)

```
┌────────────────────────────────────────┐
│  Волны врагов с нарастающей сложностью │
│  - Тестирование баланса               │
│  - Проверка комбинаций техник         │
│  - Нагрузочное тестирование           │
└────────────────────────────────────────┘
```

### Режим 3: Дуэль (план)

```
┌────────────────────────────────────────┐
│  1 на 1 с ИИ противником              │
│  - Тестирование боевого ИИ            │
│  - Проверка баланса PvP               │
│  - Тестирование блока/уклонения       │
└────────────────────────────────────────┘
```

### Режим 4: Debug (план)

```
┌────────────────────────────────────────┐
│  Отображение всей технической инфы    │
│  - Хитбоксы                           │
│  - Зоны поражения                     │
│  - Состояние ИИ                       │
│  - Логи событий                       │
└────────────────────────────────────────┘
```

## 👥 NPC Система на полигоне

### NPC Viewer - улучшения

| # | Улучшение | Статус | Описание |
|---|-----------|--------|----------|
| N.1 | Тип "Реальность" | ✅ | Отображение типа NPC (preset/temp/session) |
| N.2 | Поле "Локация" | ✅ | Где находится NPC |
| N.3 | Вкладка по локациям | 🔲 | Переключение между локациями |
| N.4 | Действия с NPC | 🔲 | Взаимодействие (разговор, торговля) |
| N.5 | Фильтр по типу | 🔲 | Фильтрация preset/temp/dynamic |

### Типы NPC (по "реальности")

| Тип | Иконка | Сохранение | Описание |
|-----|--------|------------|----------|
| **Preset** | ⭐ | БД + память | Сюжетные персонажи |
| **Session Static** | 🎮 | Сессия | Постоянные NPC локации |
| **Session Dynamic** | 📍 | До ухода | Временные NPC |
| **Temporary** | ⏳ | Только в памяти | Генерируемые монстры |
