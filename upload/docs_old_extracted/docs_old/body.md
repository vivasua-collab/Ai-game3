# 🦴 Система Тела (Body System)

**Версия:** 2.0
**Создано:** 2026-03-11
**Статус:** ✅ Реализовано

---

## 📋 Обзор

Система конструирования тел для всех сущностей мира:
- **NPC** — неигровые персонажи
- **Монстры** — враждебные существа
- **Духи** — нематериальные сущности
- **Игроки** — при генерации персонажа

### Цели системы

1. **Модульность** — сборка любого существа из компонентов
2. **Наследование** — свойства вида влияют на тело
3. **Секторные повреждения** — хитбоксы частей тела (Kenshi-style)
4. **Масштабируемость** — от мыши до титана

## 2️⃣ ВИД (Species)

### Типы видов

| Тип | Описание | Примеры |
|-----|----------|---------|
| `humanoid` | Человекоподобные | Человек, эльф, демон |
| `beast` | Звери | Волк, дракон, феникс |
| `spirit` | Духи | Призрак, элементаль |
| `hybrid` | Гибриды | Кентавр, оборотень |
| `aberration` | Аберрации | Хтонь, мутант |

### Свойства вида

- **baseStats** — диапазоны характеристик (сила, ловкость, интеллект)
- **capabilities** — может культивировать, говорить, использовать инструменты
- **aging** — продолжительность жизни, возраст зрелости
- **cultivation** — базовая ёмкость ядра, проводимость, макс. уровень

**Реализация:** `src/types/body.ts` → интерфейс `SpeciesProperties`

## 4️⃣ Система двойной HP (Kenshi-style)

### Механика

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    HP КОНЕЧНОСТИ (Пример: Рука)                          │
├─────────────────────────────────────────────────────────────────────────┤
│   ████████████████████░░░░░░░░░░░░░░░░░░░░  Красная HP (40)             │
│   │←── Функциональная HP ──→│                (паралич при 0)            │
│                                                                          │
│   ████████████████████████████████████████░░░░░░░░░░░░░░░░░░░░░░░░░░░░  │
│   │←────────── Структурная HP (80) ──────────────→│ (отрубание при 0)   │
│                                                                          │
│   Соотношение: Структурная HP = Функциональная HP × 2                    │
│   ИСКЛЮЧЕНИЕ: Сердце имеет только красную HP!                           │
└─────────────────────────────────────────────────────────────────────────┘
```

### Ключевые механики

1. **Красная HP** — функциональность. При 0 — паралич части
2. **Чёрная HP** — структурная целостность. При 0 — отрубание
3. **Кровотечение** — урон за ТИК (1 ТИК = 1 минута)
4. **Регенерация** — сначала чёрная, потом красная
5. **Приживление** — возможность приживить чужую конечность

### Сердце (особая часть)

- Только функциональная HP (80)
- Доступно для атаки при: атакующий 5+ уровень, торс в чёрной зоне
- При HP = 0 → смерть

## 7️⃣ Врождённые техники

Техники, доступные при рождении или открывающиеся при развитии.

### Структура

```typescript
interface InnateTechniqueGrant {
  techniqueId: string;      // ID техники
  unlockLevel: number;      // Уровень культивации
  mastery: number;          // Начальное мастерство (%)
}
```

### Пример: Волк

- **При рождении:** Укус (50%), Когти (40%), Вой (30%)
- **Уровень 2:** Зов стаи
- **Уровень 4:** Рык альфы
- **Эволюция:** Укус → Смертельные челюсти (ур. 5)
