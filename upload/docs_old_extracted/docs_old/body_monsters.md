## 1️⃣ Шаблоны Тела (Body Templates)

### 1.1 Типы шаблонов

| Шаблон | Описание | Используется видами |
|--------|----------|---------------------|
| `humanoid` | Двурукое двуногое | Человек, Эльф, Демон, Великан |
| `beast_quadruped` | Четвероногое | Волк, Тигр, Медведь, Дракон |
| `beast_bird` | Птица с крыльями | Орёл, Ястреб, Феникс |
| `beast_serpentine` | Змееподобное | Змея, Ламия |
| `beast_arthropod` | Членистоногое | Паук, Многоножка, Скорпион |
| `spirit` | Бесплотное | Призрак, Элементаль |

### 1.2 Классы размера

| Класс | Высота | Множитель силы | Множитель дальности |
|-------|--------|----------------|---------------------|
| `tiny` | < 30 см | 0.1x | 0.3x |
| `small` | 30-60 см | 0.3x | 0.6x |
| `medium` | 60-180 см | 1.0x | 1.0x |
| `large` | 1.8-3 м | 2.0x | 1.3x |
| `huge` | 3-10 м | 5.0x | 1.6x |

### 9.7 Интеграция в код

**Использование при расчёте урона:**
```typescript
import { 
  getWeaknessDamageMultiplier,
  getWeaknessesForSpecies,
  getResistancesForSpecies 
} from '@/data/weakness-registry';

// Расчёт урона с учётом слабостей
function calculateDamage(
  target: Entity,
  weapon: Weapon,
  baseDamage: number
): number {
  const speciesId = target.species.id;
  const material = weapon.material;
  const element = weapon.elementalDamage?.element;
  
  // Слабости
  const weaknessMult = getWeaknessDamageMultiplier(
    speciesId, 
    material, 
    element
  );
  
  // Сопротивления
  const resistances = getResistancesForSpecies(speciesId);
  let resistMult = 1.0;
  for (const resist of resistances) {
    if (resist.conditions?.element?.includes(element)) {
      resistMult *= resist.damageMultiplier;
    }
  }
  
  return baseDamage * weaknessMult * resistMult;
}
```
