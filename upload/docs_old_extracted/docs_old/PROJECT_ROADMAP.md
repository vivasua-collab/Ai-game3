## 📅 Версия 0.8.0 (Планируется - Sector System)

### 🗺️ Секторная система мира (RimWorld-style)

**Цель:** Разделение на карту мира и локальные сцены секторов.

**Концепция:**
```
World Map (50x50 секторов) → Enter Sector → Sector View (32x32 тайлов)
     ↑                                           ↓
     ←←←←←←← Exit Sector (сохранение статики) ←←←←
```

**Новое:**
- **World Map** — карта мира с секторами, фог войны, путешествия
- **Sector View** — локальная сцена Phaser при входе в сектор
- **Static Objects** — сохраняются при уходе (деревья, камни, руины)
- **Dynamic Entities** — временные NPC, генерируются при входе
- **Travel System** — перемещение между секторами с временем

**Документация:** [sector-architecture.md](./sector-architecture.md)

## 📅 Версия 0.7.0 (Текущая - Environment System)

### 🌳 Система окружения: Тестовый полигон

**Цель:** Создать систему окружения для обкатки механик перед внедрением в основную игру.

**Новое:**
- **Препятствия** — камни, овраги, скалы (физика столкновений)
- **Деревья** — ресурс + укрытие (4 типа: сосна, дуб, бамбук, духовное)
- **Рудные камни** — источники ресурсов (4 типа: железо, медь, духовная руда, кристалл)
- **Строения** — деревянные стены, двери, окна (часть карты)
- **Генератор окружения** — процедурное размещение объектов
- **NPC Viewer** — просмотр NPC на текущей локации (3-колоночный макет)

## 📅 Версия 0.6.2 (Combat Training)

### ⚔️ Боевая система: Тренировочный полигон

**Новое:**
- **Тренировочный полигон** — боевая арена с 6 мишенями
- **Метрическая система** — все расстояния в метрах (1м = 32px)
- **Система хитбоксов** — регистрация повреждений с учётом размера тела
- **Слоты техник** — 6 слотов для боевых техник (клавиши 1-6)
- **Базовый удар** — слот 1, только melee, 1м дальность, 0 Ци

### Константы хитбоксов
```
PLAYER_HITBOX_RADIUS = 24  (~0.75 м)
TARGET_HITBOX_RADIUS = 22  (~0.69 м)
METERS_TO_PIXELS = 32
```

### Формула попадания
```
effectiveDistance = distanceToCenter - hitboxRadius
attackReach = attackRange + targetHitboxRadius
hit = distanceToCenter <= attackReach
```
