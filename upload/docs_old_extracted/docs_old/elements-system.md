## ☠️ ЯД (POISON) — ОСОБАЯ СТИХИЯ

### Ограничения

> **Яд доступен ТОЛЬКО для техник типа Poison!**
> 
> Другие типы техник НЕ могут использовать poison.

### Особенности

1. **Несколько дебаффов по Grade:**
   - Common: 1 дебафф (DoT малый)
   - Refined: 2 дебаффа (DoT + Замедление)
   - Perfect: 3 дебаффа (DoT сильный + Замедление + Слабость)
   - Transcendent: 4+ дебаффа (все + Блок регенерации)

2. **Отравленное оружие:**
   - Можно нанести яд на оружие
   - Эффект при ударе (injection)
   - Ограниченное количество ударов (1-10)

3. **Без других стихийных бонусов:**
   - Яд — самостоятельный элемент
   - Не комбинируется с другими стихиями

## ⚪ НЕЙТРАЛЬНЫЙ — ОСОБЫЕ СЛУЧАИ

### Cultivation

> **Все техники культивации — neutral!**
> 
> Культивация работает с чистым Ци без стихийного окраса.

### Healing

> **Все техники исцеления — neutral!**
> 
> Исцеление — базовая техника. Эффективность зависит только от Grade и уровня.

## 🎯 Применение стихий

### 1. Техники

Каждая техника имеет **ровно одну** стихию согласно таблице ограничений.

```typescript
interface Technique {
  element: TechniqueElement; // Согласно ограничениям!
  // ...
}
```

> **Правило: Одна техника = одна стихия**
> 
> Комбинирование стихий в одной технике НЕ предусмотрено.

### 2. Оружие

Оружие может наносить урон определённой стихией (включая poison).

```typescript
interface Weapon {
  elementalDamage?: {
    element: TechniqueElement; // Включая poison!
    value: number;
  };
  // Отравленное оружие
  poisonCoating?: {
    damage: number;
    debuffs: PoisonDebuff[];
    remainingHits: number;
  };
}
```

### 3. Броня

Броня может давать сопротивление к стихии (включая poison).

```typescript
interface Armor {
  elementalResistance?: Partial<Record<TechniqueElement, number>>;
  // poisonResistance включён в TechniqueElement
}
```

### 4. NPC

NPC могут иметь врождённую стихию (включая poison для ядовитых существ).

```typescript
interface NPC {
  innateElement?: TechniqueElement; // Включая poison!
}
```

### Камни Ци — БЕЗ стихии!

> **Камни Ци НЕ имеют стихийного окраса.**
> 
> Это чистая энергия, которую практик может преобразовать в любую стихию через технику.
> 
> ```typescript
> interface QiStone {
>   capacity: number;  // Объём Ци
>   quality: number;   // Качество
>   // НЕТ поля element!
> }
> ```

### Техники яда (poison) — ОСОБАЯ СТИХИЯ

> **☠️ Яд имеет уникальную механику нескольких дебаффов!**

| Grade | Дебафф 1 | Дебафф 2 | Дебафф 3 | Дебафф 4 |
|-------|----------|----------|----------|----------|
| Common | DoT (малый) | — | — | — |
| Refined | DoT | Замедление | — | — |
| Perfect | DoT (сильный) | Замедление | Слабость | — |
| Transcendent | DoT (max) | Замедление | Слабость+ | Блок регена |

**Типы дебаффов яда:**
- **DoT** — урон по времени (body или qi)
- **Замедление** — -% скорости
- **Слабость** — -% урона/защиты
- **Блок регена** — нет восстановления HP/Ци
- **Паралич** — невозможность действия
- **Некроз** — постоянная потеря HP
