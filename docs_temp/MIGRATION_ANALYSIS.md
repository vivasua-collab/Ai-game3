## 🔬 Детальный анализ переносимости

### ✅ Частично переносимый код (~30,000 строк)

**Переносимая логика:**
- Алгоритмы расчёта урона (damage-calculator.ts)
- Формулы развития характеристик
- Логика генерации техник
- Генераторы имён и описаний
- Конфигурации данных (data/*.ts)

**Проблемы при миграции:**
1. **Серверная архитектура** — весь код в `lib/game/server/` завязан на API
2. **Prisma модели** — требуют адаптации под SQLite Unity
3. **Типы** — требуют удаления Next.js-специфичных полей

### ❌ Непереносимый код (~91,000 строк)

| Категория | Причина |
|-----------|---------|
| **Phaser код** | Полная несовместимость с Unity |
| **React компоненты** | Unity UI = другая система |
| **API routes** | Не нужны в single-player |
| **Server actions** | Не нужны в single-player |
| **Prisma queries** | Другая ORM в Unity |
